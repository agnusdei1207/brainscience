# FET (전계효과 트랜지스터)

## 정의

**FET(Field Effect Transistor)**는 **전계(전압)에 의해 제어되는 단극성 트랜지스터**다. 게이트 전압으로 채널의 전도도를 제어하며, BJT와 달리 전류 제어가 아닌 **전압 제어 방식**이다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    FET의 기본 구조                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   기본 구조:                                                   │
│                                                                 │
│                    Drain (D)                                    │
│                       │                                         │
│              ┌────────┴────────┐                               │
│              │                 │                               │
│              │     N형 또는     │                               │
│              │     P형 채널     │                               │
│              │                 │                               │
│   Gate (G) ──┤    [게이트]     │                               │
│              │   (전계 형성)    │                               │
│              │                 │                               │
│              │                 │                               │
│              └────────┬────────┘                               │
│                       │                                         │
│                    Source (S)                                   │
│                                                                 │
│   특징:                                                         │
│   • 게이트 전압(Vgs)으로 드레인 전류(Id) 제어                   │
│   • 단극성: 전자 또는 정공 중 하나만 캐리어                     │
│   • 높은 입력 임피던스 (MΩ~GΩ)                                  │
│   • 전압 제어 소자                                              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. FET 분류

```
┌─────────────────────────────────────────────────────────────────┐
│                      FET 분류                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│                        FET                                      │
│                         │                                       │
│           ┌─────────────┴─────────────┐                         │
│           │                           │                         │
│         JFET                       MOSFET                       │
│    (Junction FET)         (Metal-Oxide-Semiconductor FET)      │
│           │                           │                         │
│      ┌────┴────┐               ┌─────┴─────┐                    │
│      │         │               │           │                    │
│   N-채널   P-채널           E-MOSFET   D-MOSFET                 │
│   JFET     JFET         (Enhancement) (Depletion)               │
│                                 │           │                    │
│                            ┌────┴───┐   ┌───┴────┐              │
│                            │        │   │        │              │
│                          N-type  P-type N-type P-type            │
│                                                                 │
│   JFET:                                                         │
│   • 게이트-채널 PN 접합                                          │
│   • 공핍 모드만 동작                                            │
│   • 높은 입력 임피던스                                          │
│   • 저잡음 증폭기                                               │
│                                                                 │
│   MOSFET:                                                       │
│   • 게이트-채널 산화막 절연                                      │
│   • 매우 높은 입력 임피던스                                      │
│   • 디지털 IC의 핵심 소자                                        │
│   • CMOS = NMOS + PMOS                                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. MOSFET 동작 원리

```
┌─────────────────────────────────────────────────────────────────┐
│                N채널 Enhancement MOSFET                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   구조:                                                         │
│                                                                 │
│                    D (Drain)                                    │
│                     │                                           │
│               ┌─────┴─────┐                                     │
│               │     N+    │                                     │
│               ├───────────┤   G (Gate)                         │
│               │           │    │                                │
│               │    P형     │ ───┴─── [게이트 산화막 SiO₂]        │
│               │  (기판)    │                                     │
│               │           │                                     │
│               ├───────────┤                                     │
│               │     N+    │                                     │
│               └─────┬─────┘                                     │
│                     │                                           │
│                    S (Source)                                   │
│                                                                 │
│   동작:                                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │  Vgs = 0 (OFF):    Vgs > Vth (ON):                      │ │
│   │                                                         │ │
│   │    D        S         D        S                         │ │
│   │    │        │         │        │                         │ │
│   │   N+       N+        N+       N+                         │ │
│   │    │        │    │   │        │    │                     │ │
│   │    │   P    │    │   │   ▶▶▶  │    │                     │ │
│   │    │ (채널X)│    │   │ (채널) │    │                     │ │
│   │    │        │         │        │                         │ │
│   │   N+       N+        N+       N+                         │ │
│   │                                                         │ │
│   │  전류 차단          전류 흐름 (Id)                        │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   Vgs > Vth (임계 전압) → 반전층 채널 형성 → 전류 흐름           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. FET 특성 곡선

```
┌─────────────────────────────────────────────────────────────────┐
│                   FET 출력 특성                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Id (드레인 전류)                                              │
│     ↑                                                           │
│     │     Vgs4                                                 │
│     │     ┌───────────────────────────────────                │
│     │     │                                                   │
│     │     │  Vgs3                                             │
│     │     │  ┌────────────────────────────                    │
│     │     │  │                                                │
│     │     │  │  Vgs2                                          │
│     │     │  │  ┌────────────────────                          │
│     │     │  │  │                                             │
│     │     │  │  │  Vgs1 (= Vth)                                │
│     │     │  │  │  ┌────────────                               │
│     │     │  │  │  │                                          │
│   0 ─┼─────┼──┼──┼──┼─────────────────────────────▶ Vds       │
│     │     │  │  │  │                                          │
│     │     │  │  │  │                                          │
│     │     [저항 영역]  [포화 영역]                              │
│     │                                                         │
│     │  저항 영역: Id ∝ Vds (선형)                              │
│     │  포화 영역: Id ≈ 일정 (Vds 무관)                          │
│     │                                                         │
│   전류 방정식 (포화 영역):                                      │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  Id = (½) × Kn × (Vgs - Vth)² × (1 + λVds)               │ │
│   │                                                         │ │
│   │  Kn: 전도도 파라미터                                     │ │
│   │  Vth: 임계 전압 (Threshold Voltage)                      │ │
│   │  λ: 채널 길이 변조 파라미터                               │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 상세 설명

### JFET vs MOSFET

```
┌─────────────────────────────────────────────────────────────────┐
│                 JFET vs MOSFET 비교                             │
├─────────────────┬─────────────────┬─────────────────────────────┤
│     항목        │      JFET       │         MOSFET              │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   게이트 구조    │  PN 접합        │   산화막 (SiO₂) 절연        │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   입력 임피던스  │  높음 (MΩ)      │   매우 높음 (GΩ~TΩ)         │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   동작 모드      │  공핍만         │   공핍/강화                  │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   집적도        │  낮음           │   높음                      │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   제조 복잡도    │  낮음           │   높음                      │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   정전 파괴      │  취약           │   강함                      │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   주요 용도      │  RF 증폭기      │   디지털 IC, 전력 제어      │
│                 │  저잡음 증폭     │   아날로그 스위치            │
└─────────────────┴─────────────────┴─────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **FET**: 전계(전압)에 의해 제어되는 단극성 트랜지스터
> **MOSFET**: 산화막으로 게이트가 절연된 FET, 디지털 IC의 핵심

### 2. 빈출 키워드

- **전압 제어**: Vgs로 Id 제어
- **단극성**: 전자 또는 정공 하나만 캐리어
- **임계 전압 (Vth)**: 채널 형성 시작 전압
- **CMOS**: NMOS + PMOS, 저전력
