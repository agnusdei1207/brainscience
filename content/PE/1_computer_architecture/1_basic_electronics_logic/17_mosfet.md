# MOSFET

## 정의

**MOSFET(Metal-Oxide-Semiconductor FET)**은 게이트와 채널 사이에 **산화막(SiO₂)을 배치**하여 게이트 전압으로 채널의 전도도를 제어하는 FET다. 디지털 IC의 핵심 소자다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    MOSFET 구조                              │
│                                                                 │
│   NMOS (N채널):                                                   │
│                                                                 │
│                      D (Drain)                               │
│                       │                                     │
│              N+       │         N+                       │
│              ┌────────┴─────────────────────────┐              │
│              │                                     │              │
│   G ────────┤  SiO₂  (게이트 산화막)            │              │
│              │         │                              │              │
│              └─────────────────────────────────┘              │
│                      P-substrate                         │
│                           │                                     │
│                      S (Source)                             │
│                                                                 │
│   PMOS (P채널):                                                   │
│                                                                 │
│                      D (Drain)                              │
│                       │                                     │
│              P+       │         P+                       │
│              └────────┴─────────────────────────┐              │
│              │                                     │              │
│   G ────────┤  SiO₂  (게이트 산화막)            │              │
│              │         │                              │              │
│              └─────────────────────────────────┘              │
│                      N-substrate                         │
│                           │                                     │
│                      S (Source)                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. MOSFET 동작 원리

```
┌─────────────────────────────────────────────────────────────────┐
│                 Enhancement MOSFET (E-MOSFET)                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   NMOS 특성:                                                   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐│
│   │  Vgs        │  상태           │  Id                  │ │
│   ├────────────┼────────────────┼────────────────────────┤ │
│   │  < Vth      │  차단 (OFF)     │  Id ≈ 0              │ │
│   │  > Vth      │  활성 (ON)      │  Id ∝ (Vgs - Vth)       │ │
│   └────────────┴────────────────┴────────────────────────┘ │
│                                                                 │
│   PMOS 특성:                                                   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐│
│   │  Vgs        │  상태           │  Id                  │ │
│   ├────────────┼────────────────┼────────────────────────┤ │
│   │  > Vtp      │  차단 (OFF)     │  Id ≈ 0              │ │
│   │  < Vtp      │  활성 (ON)      │  Id ∝ (Vtp - Vgs)       │ │
│   └────────────┴────────────────┴────────────────────────┘ │
│                                                                 │
│   Vth (Threshold Voltage): 0.5V ~ 1.0V (공정에 따라 다름)          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. MOSFET 특성 곡선

```
┌─────────────────────────────────────────────────────────────────┐
│                   NMOS 특성 곡선                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Id │                                                       │
│     ▲                                                        │
│     │        ┌────────────────────────────────────┐          │
│     │        │       포화 영역 (Saturation)       │          │
│     │        │        Id = Id(sat)              │          │
│     │        └────────────────────────────────────┘          │
│     │        │                                        │          │
│     │        │   선형 영역 (Linear/Triode)              │          │
│     │        │   Id = K × (Vgs - Vth) × Vds             │          │
│     │        │                                        │          │
│     │        └────────────────────────────────────┘          │
│     │                                                 │          │
│     │        Vth                                    Vgs        │
│     └─────────────────────────────────────────────────────▶ Vgs   │
│                                                                 │
│   영역:                                                          │
│   • 차단 (Cutoff): Vgs < Vth, Id ≈ 0                              │
│   • 선형 (Linear): Vgs > Vth, Vds > Vgs - Vth                    │
│   • 포화 (Saturation): Vds < Vgs - Vth, Id = Id(sat)             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 상세 설명

### FinFET (핀펫)

```
┌─────────────────────────────────────────────────────────────────┐
│                    FinFET 구조                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   평면 MOSFET vs FinFET:                                        │
│                                                                 │
│   평면 MOSFET:              FinFET (3D):                        │
│   ┌─────────────┐         ┌─────────────┐                     │
│   │             │         │    Gate     │                     │
│   │   Gate      │         │  ┌───────┐  │                     │
│   │ ─────────── │         │  │ Fin   │  │                     │
│   │             │         │  │       │  │                     │
│   │   채널      │         │  │ 채널  │  │                     │
│   │             │         │  │       │  │                     │
│   │             │         │  │ Fin   │  │                     │
│   │             │         │  └───────┘  │                     │
│   └─────────────┘         └─────────────┘                     │
│                                                                 │
│   FinFET 장점:                                                 │
│   • 향상된 게이트 제어: 3면 게이트로 채널 감싸기                   │
│   • 단채널 효과 감소: 더 짧은 채널 가능                         │
│   • 낮은 누설 전류: 향상된 게이트 제어                          │
│   • 높은 성능: 22nm 이하 공정에서 필수                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **MOSFET**: 산화막으로 게이트가 절연된 전계효과 트랜지스터
> **FinFET**: 핀 형태 게이트로 3D 구조를 가진 MOSFET

### 2. 빈출 키워드

- **Vth (임계 전압)**: 채널 형성 시작 전압
- **선형/포화 영역**: 증폭/스위칭 동작
- **NMOS/PMOS**: N채널/P채널 MOSFET
- **FinFET**: 22nm 이하 공정의 핵심 기술
