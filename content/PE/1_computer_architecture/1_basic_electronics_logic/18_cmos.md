# CMOS (Complementary MOS)

## 정의

**CMOS(Complementary MOS)**는 **NMOS와 PMOS를 상보적으로 조합**하여 구성한 논리 회로 기술이다. 정적 상태에서 거의 전력을 소모하지 않아 **저전력 디지털 회로의 핵심 기술**이다.

```
┌─────────────────────────────────────────────────────────────────┐
│                   CMOS 인버터 구조                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│                      VDD                                       │
│                       │                                        │
│              ┌────────┴────────┐                               │
│              │      PMOS       │                               │
│              │     (P채널)     │                               │
│              └────────┬────────┘                               │
│                       │                                        │
│       In ─────────────┼────────────────────▶ Out               │
│                       │                                        │
│              ┌────────┴────────┐                               │
│              │      NMOS       │                               │
│              │     (N채널)     │                               │
│              └────────┬────────┘                               │
│                       │                                        │
│                      GND                                       │
│                                                                 │
│   동작:                                                         │
│   • In = 1 (High): PMOS OFF, NMOS ON → Out = 0 (Low)           │
│   • In = 0 (Low): PMOS ON, NMOS OFF → Out = 1 (High)           │
│                                                                 │
│   정적 상태에서 두 트랜지스터 중 하나는 항상 OFF                 │
│   → VDD에서 GND로 직접 전류 경로 없음 → 전력 소모 최소화        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. CMOS의 장점

```
┌─────────────────────────────────────────────────────────────────┐
│                   CMOS의 핵심 장점                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ① 저전력 소모                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │  정적 전력 (Static Power): P_static = VDD × I_leak       │ │
│   │  • 누설 전류만 흐름 (매우 작음, nA~μA 수준)              │ │
│   │                                                         │ │
│   │  동적 전력 (Dynamic Power): P_dyn = C × VDD² × f        │ │
│   │  • 상태 전환 시에만 소모                                  │ │
│   │  • 충/방전 전류                                           │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ② 높은 잡음 내성                                              │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │  NMH (High Noise Margin): VDD - VIH_min                 │ │
│   │  NML (Low Noise Margin): VIL_max - 0                    │ │
│   │                                                         │ │
│   │  CMOS: NMH ≈ NML ≈ VDD/2 (우수한 잡음 내성)             │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ③ 단일 전원 공급                                              │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │  NMOS만: VDD + VBB (서브스트레이트 바이어스) 필요        │ │
│   │  CMOS: VDD만 필요 (단일 전원)                            │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ④ 높은 팬아웃                                                │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │  CMOS 입력: 게이트 산화막만 (매우 높은 임피던스)          │ │
│   │  → 많은 게이트를 구동 가능                                │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. CMOS 전력 소모

```
┌─────────────────────────────────────────────────────────────────┐
│                   CMOS 전력 소모 분석                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   총 전력 = 정적 전력 + 동적 전력                                 │
│                                                                 │
│   P_total = P_static + P_dynamic                               │
│           = VDD × I_leak + α × C × VDD² × f                    │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │  정적 전력 (누설 전류):                                   │ │
│   │  • 서브스레시 누설 (Subthreshold Leakage)                │ │
│   │  • 게이트 누설 (Gate Leakage)                            │ │
│   │  • 접합 누설 (Junction Leakage)                          │ │
│   │  • VDD 감소로 누설 감소                                   │ │
│   │                                                         │ │
│   │  동적 전력 (스위칭):                                      │ │
│   │  P = C_L × VDD² × f                                      │ │
│   │                                                         │ │
│   │  • C_L: 부하 커패시턴스                                   │ │
│   │  • VDD: 전원 전압 (제곱에 비례, 가장 큰 영향)             │ │
│   │  • f: 동작 주파수                                         │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   저전력 설계 기법:                                             │
│   • VDD 스케일링: 전압 낮춤 → 전력 2차 감소                      │
│   • 클럭 게이팅: 사용하지 않는 모듈 클럭 차단                     │
│   • 전력 게이팅: 누설 전류 경로 차단                              │
│   • DVFS: 동적 전압/주파수 스케일링                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. CMOS 게이트

```
┌─────────────────────────────────────────────────────────────────┐
│                   기본 CMOS 게이트                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ① NAND 게이트 (2입력):                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                      VDD                                 │ │
│   │                       │                                  │ │
│   │              ┌────────┴────────┐                         │ │
│   │              │    PMOS A       │                         │ │
│   │              └────────┬────────┘                         │ │
│   │                       │                                  │ │
│   │              ┌────────┴────────┐                         │ │
│   │              │    PMOS B       │                         │ │
│   │              └────────┬────────┘                         │ │
│   │                       │                                  │ │
│   │   A ─────────────────┼────────────────────▶ Out          │
│   │   B ─────────────────┼────────────────────▶              │
│   │                       │                                  │ │
│   │              ┌────────┴────────┐                         │ │
│   │              │    NMOS A       │                         │ │
│   │              └────────┬────────┘                         │ │
│   │                       │                                  │ │
│   │              ┌────────┴────────┐                         │ │
│   │              │    NMOS B       │                         │ │
│   │              └────────┬────────┘                         │ │
│   │                       │                                  │ │
│   │                      GND                                 │ │
│   │                                                         │ │
│   │   진리표: Y = NOT(A AND B) = (A·B)'                      │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ② NOR 게이트 (2입력):                                        │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   Y = NOT(A OR B) = (A+B)'                              │ │
│   │   PMOS 직렬, NMOS 병렬 구성                              │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ③ 복합 게이트:                                               │
│   • AND: NAND + Inverter                                       │
│   • OR: NOR + Inverter                                         │
│   • XOR: NAND/NOR 조합                                         │
│   • Full Adder: XOR + AND 조합                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **CMOS**: NMOS와 PMOS를 상보적으로 조합한 저전력 논리 회로
> **정적 전력**: 누설 전류로 인한 대기 전력

### 2. 빈출 키워드

- **저전력**: 정적 상태에서 전력 소모 거의 없음
- **잡음 내성**: VDD/2 수준의 우수한 마진
- **동적 전력**: P = CV²f
- **NAND/NOR**: CMOS 기본 게이트

### 3. 예상 문제

**Q1. CMOS가 저전력인 이유를 설명하시오.**

**Q2. CMOS의 동적 전력 공식과 각 변수의 영향을 기술하시오.**

**Q3. CMOS NAND 게이트의 동작 원리를 설명하시오.**
