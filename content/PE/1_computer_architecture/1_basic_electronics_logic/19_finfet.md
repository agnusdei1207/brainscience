# 핀펫 (FinFET)

## 정의

**FinFET(Fin Field Effect Transistor)**는 게이트가 **핀(Fin) 형태의 채널을 3면으로 감싸는** 3D 구조의 MOSFET다. 22nm 이하 공정에서 단채널 효과를 극복하기 위해 개발되었다.

```
┌─────────────────────────────────────────────────────────────────┐
│               평면 MOSFET vs FinFET 구조 비교                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   평면 MOSFET:                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                       Gate                               │ │
│   │                    ─────────────                         │ │
│   │                        │                                  │ │
│   │                    SiO₂ 산화막                             │ │
│   │                        │                                  │ │
│   │   ─────────────────────────────────────                  │ │
│   │              채널 (평면)                                   │ │
│   │   ─────────────────────────────────────                  │ │
│   │                                                         │ │
│   │   게이트가 채널 위쪽 면만 제어                             │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   FinFET (3D):                                                  │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                       Gate                               │ │
│   │                    ─────────────                         │ │
│   │                    │    │    │                           │ │
│   │                    │ ┌──┴──┐ │                           │ │
│   │                    │ │ Fin │ │ ← 3면 게이트                │ │
│   │                    │ └──┬──┘ │                           │ │
│   │                    │    │    │                           │ │
│   │                                                         │ │
│   │   • Fin: 핀 형태 실리콘 구조                              │ │
│   │   • 게이트가 핀의 3면을 감쌈                               │ │
│   │   • 향상된 게이트 제어                                    │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **FinFET**: 핀 형태 채널을 게이트가 3면으로 감싸는 3D 트랜지스터
> **GAA**: 게이트가 채널 4면을 감싸는 차세대 구조

### 2. 빈출 키워드

- **3면 게이트**: 향상된 채널 제어
- **단채널 효과 감소**: FinFET의 핵심 장점
- **저누설 전류**: 향상된 게이트 제어로 달성
- **22nm 이하**: FinFET이 필수인 공정 영역
