# GAA (Gate-All-Around)

## 정의

**GAA(Gate-All-Around)**는 게이트가 채널의 **4면을 모두 감싸는** 차세대 트랜지스터 구조다. FinFET의 3면 제어를 4면으로 확장하여 더 완벽한 채널 제어를 제공한다. 3nm 이하 공정에서 핵심 기술이다.

```
┌─────────────────────────────────────────────────────────────────┐
│           FinFET vs GAA 구조 비교                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   FinFET (3면 게이트):                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │              Gate (위)                                   │ │
│   │                ↓                                         │ │
│   │         ┌─────────────┐                                  │ │
│   │   Gate  │             │  Gate                            │ │
│   │   (좌)  │     핀      │  (우)                             │ │
│   │   ───▶  │   (채널)    │  ◀───                            │ │
│   │         │             │                                  │ │
│   │         └─────────────┘                                  │ │
│   │                                                         │ │
│   │   게이트 3면: 위, 좌, 우                                   │ │
│   │   아래면: 기판 (Si)와 접촉                                 │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   GAA (4면 게이트):                                             │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │           ┌── Gate (위) ──┐                             │ │
│   │           │               │                             │ │
│   │    Gate   │    나노시트    │   Gate                       │ │
│   │    (좌)   │    (채널)     │   (우)                        │ │
│   │           │               │                             │ │
│   │           └── Gate (아래) ─┘                             │ │
│   │                                                         │ │
│   │   게이트 4면: 위, 아래, 좌, 우 (완전한 감싸기)              │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. GAA의 장점

```
┌─────────────────────────────────────────────────────────────────┐
│                    GAA의 핵심 장점                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ① 최적의 게이트 제어                                          │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  • 4면 게이트로 채널 완전 감싸기                          │ │
│   │  • 채널 전체의 전하 제어 가능                               │ │
│   │  • 누설 전류 경로 완전 차단                                │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ② 누설 전류 획기적 감소                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  • FinFET 대비 2배 이상 감소 가능                         │ │
│   │  • 서브스레시 누설(Swing) 감소                            │ │
│   │  • 저전력 소자 구현                                       │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ③ 성능 향상                                                  │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  • 높은 전류 구동 능력(Imax)                              │ │
│   │  • 낮은 Vth 달성 가능                                     │ │
│   │  • 스위칭 속도 향상                                       │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ④ 스케일링 연장                                               │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  • 3nm, 2nm, 1nm 공정까지 확장 가능                       │ │
│   │  • FinFET 물리적 한계 극복                                │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. GAA 구현 방식

```
┌─────────────────────────────────────────────────────────────────┐
│                   GAA 구현 기술                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │

│   ┌─────────────────────────────────────────────────────────┐ │

│   │   장점: 높은 전류 밀도, 우수한 구동력                      │ │
│   │   적용: Samsung 3nm MBCFET                               │ │
│   │                                                         │ │
│   │   ┌───────────────────────────────────────────┐         │ │
│   │   │  ═══ Gate ═══                              │         │ │
│   │   │     │ Sheet │     ← 넓은 채널               │         │ │
│   │   │  ═══ Gate ═══                              │         │ │
│   │   └───────────────────────────────────────────┘         │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ② Forksheet FET:                                             │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   구조: 포크 형태로 분리된 시트                            │ │
│   │   장점: NMOS/PMOS 분리로 최적화                          │ │
│   │   적용: Intel 20A, 18A                                   │ │
│   │                                                         │ │
│   │      NMOS          PMOS                                  │ │
│   │     ┌───┐         ┌───┐                                  │ │
│   │     │   │         │   │                                  │ │
│   │     │   │         │   │                                  │ │
│   │     └───┘         └───┘                                  │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ③ CFET (Complementary FET):                                  │
│   ┌─────────────────────────────────────────────────────────┐ │

│   │       ┌───── PMOS ─────┐                                 │ │
│   │       │                 │                                 │ │
│   │       ├───── NMOS ─────┤  ← 수직 적층                    │ │
│   │       │                 │                                 │ │
│   │       └─────────────────┘                                 │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **GAA**: 게이트가 채널 4면을 감싸는 구조, FinFET 진화형
> **Nanosheet**: 시트 형태 채널을 사용하는 GAA 변형

### 2. 빈출 키워드

- **4면 게이트**: 완전한 채널 제어
- **누설 전류 감소**: FinFET 대비 향상
- **3nm 이하**: GAA가 필요한 공정 영역
- **CFET**: NMOS/PMOS 수직 적층
