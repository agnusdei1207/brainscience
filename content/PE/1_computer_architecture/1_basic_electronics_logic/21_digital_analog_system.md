# 디지털 시스템 vs 아날로그 시스템

## 정의

**디지털 시스템**은 연속적인 신호를 **이산적인 값(0과 1)**으로 표현하고 처리하는 시스템이다. **아날로그 시스템**은 연속적인 물리량을 그대로 처리한다.

```
┌─────────────────────────────────────────────────────────────────┐
│              아날로그 vs 디지털 신호 비교                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   아날로그 신호:                    디지털 신호:                │
│                                                                 │
│   ▲                                ▲                           │
│   │     ~~~                        │                           │
│   │   ~~   ~~~                     │   ┌───┐   ┌───┐          │
│   │  ~       ~~~                   │   │   │   │   │           │
│   │ ~          ~~~                 │───┘   └───┘   └───       │
│   │~             ~~~               │                           │
│   └──────────────────────▶ 시간    └──────────────────────▶ 시간│
│                                                                 │
│   • 연속적                          • 이산적                    │
│   • 무한한 값                        • 유한한 값 (0, 1)          │
│   • 물리량 직접 표현                 • 2진수로 코딩              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 아날로그 시스템 특성

```
┌─────────────────────────────────────────────────────────────────┐
│                   아날로그 시스템                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   장점:                                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │ • 연속적 정보 처리 (자연스러운 신호)                        │ │
│   │ • 무한 해상도                                            │ │
│   │ • 단순한 회로 (일부 응용)                                  │ │
│   │ • 낮은 지연 시간                                          │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   단점:                                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │ • 잡음 취약성 (Noise Susceptibility)                     │ │
│   │ • 신호 감쇠 (Attenuation)                                 │ │
│   │ • 왜곡 (Distortion)                                       │ │
│   │ • 저장/처리 어려움                                        │ │
│   │ • 정확한 복원 불가능 (오차 누적)                           │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   응용:                                                         │
│   • 오디오 앰프 (진공관, 트랜지스터 선형 증폭)                   │
│   • RF 통신 (변조/복조)                                         │
│   • 센서 신호 처리                                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 디지털 시스템 특성

```
┌─────────────────────────────────────────────────────────────────┐
│                   디지털 시스템                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   장점:                                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │ • 잡음 내성 (Noise Immunity)                             │ │
│   │   - 0/1 구분만 하면 되므로 잡음에 강함                     │ │
│   │   - 재생(Regeneration) 가능                              │ │
│   │                                                         │ │
│   │ • 정확한 저장/처리                                        │ │
│   │   - 오차 누적 없음                                        │ │
│   │   - 완벽한 복사 가능                                      │ │
│   │                                                         │ │
│   │ • 프로그래밍 가능                                          │ │
│   │   - 소프트웨어로 기능 변경                                 │ │
│   │   - 범용성                                                │ │
│   │                                                         │ │
│   │ • 암호화 가능                                              │ │
│   │   - 보안 통신                                             │ │
│   │   - 인증                                                  │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   단점:                                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │ • 양자화 오류 (Quantization Error)                       │ │
│   │ • 샘플링 제약 (Nyquist 한계)                              │ │
│   │ • A/D 변환 필요                                           │ │
│   │ • 높은 대역폭 요구                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. A/D 및 D/A 변환

```
┌─────────────────────────────────────────────────────────────────┐
│                아날로그-디지털 변환 과정                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   [아날로그 입력] ──▶ [샘플링] ──▶ [양자화] ──▶ [부호화] ──▶ [디지털]│
│                                                                 │
│   ① 샘플링 (Sampling):                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   연속 신호 → 이산 시간 신호                              │ │
│   │   fs ≥ 2 × fmax  (Nyquist 정리)                         │ │
│   │                                                         │ │
│   │   ▲                    ▲                                │ │
│   │   │    ~~~             │  ●   ●                         │ │
│   │   │  ~~   ~~~          │    ●    ●                      │ │
│   │   │ ~      ~~~    →    │      ●    ●                    │ │
│   │   └─────────▶          └──────────▶                     │ │
│   │    연속                  샘플링                           │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ② 양자화 (Quantization):                                     │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   연속 진폭 → 이산 진폭 레벨                              │ │
│   │   양자화 오류: Qe ≤ LSB/2                                │ │
│   │                                                         │ │
│   │   ▲                                                    │ │
│   │   │      ●───┐                                         │ │
│   │   │    ●     │  ← 양자화 레벨                            │ │
│   │   │  ●       │                                          │ │
│   │   │●         │                                          │ │
│   │   └──────────────▶                                      │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ③ 부호화 (Encoding):                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   양자화 레벨 → 2진 코드                                  │ │
│   │   예: 8-bit ADC → 256 레벨 (0~255)                      │ │
│   │                                                         │ │
│   │   레벨 0  → 00000000                                    │ │
│   │   레벨 1  → 00000001                                    │ │
│   │   레벨 127 → 01111111                                    │ │
│   │   레벨 255 → 11111111                                    │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **디지털 시스템**: 이산적인 값(0, 1)으로 정보를 표현/처리
> **양자화 오류**: 연속 신호를 이산 레벨로 변환 시 발생하는 오차

### 2. 빈출 키워드

- **잡음 내성**: 디지털의 핵심 장점
- **Nyquist 정리**: fs ≥ 2fmax
- **양자화**: 연속→이산 진폭 변환
- **재생**: 디지털 신호의 완벽한 복원

### 3. 예상 문제

**Q1. 디지털 시스템이 아날로그 시스템보다 잡음에 강한 이유를 설명하시오.**

**Q2. A/D 변환 과정을 3단계로 설명하시오.**

**Q3. 양자화 오류를 최소화하는 방법을 기술하시오.**
