# 카르노 맵 (Karnaugh Map)

## 정의

**카르노 맵(Karnaugh Map, K-Map)**은 1953년 Maurice Karnaugh가 개발한 **논리 함수를 시각적으로 최소화**하는 방법이다. 진리표를 2차원 격자로 표현하여 인접한 1들을 그룹화함으로써 최소 논리식을 도출한다.

```
┌─────────────────────────────────────────────────────────────────┐
│              카르노 맵 기본 구조 (2, 3, 4변수)                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   2변수 카르노 맵:                                               │
│           B                                                    │
│        0     1                                                 │
│   ┌─────────────┐                                              │
│  A│ 0  │  m1  │     m0 = A'B'                                 │
│   ├─────┼──────┤     m1 = A'B                                  │
│   │ 1  │  m3  │     m2 = AB'                                   │
│   └─────────────┘     m3 = AB                                   │
│       m0   m2                                                 │
│                                                                 │
│   3변수 카르노 맵:                                               │
│           BC                                                   │
│       00  01  11  10                                           │
│   ┌───────────────────┐                                        │
│  A│ 0 │ 1 │ 3 │ 2 │     그레이 코드 순서!                       │
│   ├───┼───┼───┼───┤                                            │
│   │ 4 │ 5 │ 7 │ 6 │                                            │
│   └───────────────────┘                                        │
│                                                                 │
│   4변수 카르노 맵:                                               │
│           CD                                                   │
│       00  01  11  10                                           │
│   ┌───────────────────┐                                        │
│AB│   │   │   │   │                                             │
│ 00│ 0 │ 1 │ 3 │ 2 │                                             │
│ 01│ 4 │ 5 │ 7 │ 6 │                                             │
│ 11│12 │13 │15 │14 │                                             │
│ 10│ 8 │ 9 │11 │10 │                                             │
│   └───────────────────┘                                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 그레이 코드 순서

```
┌─────────────────────────────────────────────────────────────────┐
│                카르노 맵의 그레이 코드 배치                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   💡 핵심: 인접한 셀은 1비트만 다르다                            │
│                                                                 │
│   2비트 그레이 코드:                                             │
│   ┌─────────────────────────────┐                              │
│   │ 00 → 01 → 11 → 10          │                              │
│   │  (0)  (1)  (3)  (2)         │                              │
│   └─────────────────────────────┘                              │
│        ↑_____________↑                                         │
│        순환 (10과 00도 인접)                                    │
│                                                                 │
│   3비트 그레이 코드:                                             │
│   ┌─────────────────────────────┐                              │
│   │ 000 → 001 → 011 → 010      │                              │
│   │   ↓                    ↑    │                              │
│   │ 100 → 101 → 111 → 110      │                              │
│   └─────────────────────────────┘                              │
│                                                                 │
│   이유: 인접 셀끼리 묶으면 변수 1개 제거                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 그룹화 규칙

```
┌─────────────────────────────────────────────────────────────────┐
│                 카르노 맵 그룹화 규칙                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ① 그룹 크기: 1, 2, 4, 8, 16... (2의 거듭제곱)                 │
│                                                                 │
│   ② 그룹 형태: 직사각형 또는 정사각형                            │
│                                                                 │
│   ③ 그룹 조건: 1인 셀만 포함 (0은 포함 불가)                     │
│                                                                 │
│   ④ 경계 순환: 맵의 양 끝이 연결됨                               │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │         │                                              │ │
│   │    ┌────┼────┐    좌우 연결                             │ │
│   │    │ 1  │ 1  │    위아래 연결                           │ │
│   │    │    │    │    모서리 연결 가능                      │ │
│   │    └────┼────┘                                         │ │
│   │         │                                              │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ⑤ 겹침 허용: 같은 셀이 여러 그룹에 속할 수 있음                │
│                                                                 │
│   ⑥ 최대 그룹: 가능한 큰 그룹으로 묶기                          │
│                                                                 │
│   ⑦ 최소 그룹: 모든 1을 커버하는 최소 그룹 수                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 최소화 예시

```
┌─────────────────────────────────────────────────────────────────┐
│              3변수 카르노 맵 최소화 예시                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   F(A,B,C) = Σm(1, 3, 5, 7)                                    │
│                                                                 │
│           BC                                                   │
│       00  01  11  10                                           │
│   ┌───────────────────┐                                        │
│  A│ 0 │ 1 │ 1 │ 0 │                                             │
│   ├───┼───┼───┼───┤                                            │
│   │ 0 │ 1 │ 1 │ 0 │                                             │
│   └───────────────────┘                                        │
│        ↑_______↑                                                │
│         그룹 1 (C=1인 열)                                       │
│                                                                 │
│   그룹 분석:                                                    │
│   - 열 01, 11 묶음 → B=1, C=1인데...                            │
│   - A가 0이든 1이든 상관없음 → A 제거                            │
│   - 결과: F = C                                                 │
│                                                                 │
│   논리식 확인:                                                  │
│   m1 = A'B'C, m3 = A'BC, m5 = AB'C, m7 = ABC                   │
│   F = A'B'C + A'BC + AB'C + ABC                                │
│     = A'C(B'+B) + AC(B'+B)                                     │
│     = A'C + AC = C(A'+A) = C                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│              4변수 카르노 맵 최소화 예시                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   F(A,B,C,D) = Σm(0, 2, 8, 10)                                 │
│                                                                 │
│           CD                                                   │
│       00  01  11  10                                           │
│   ┌───────────────────┐                                        │
│AB│   │   │   │   │                                             │
│ 00│ 1 │ 0 │ 0 │ 1 │   ← 모서리 그룹!                            │
│ 01│ 0 │ 0 │ 0 │ 0 │                                             │
│ 11│ 0 │ 0 │ 0 │ 0 │                                             │
│ 10│ 1 │ 0 │ 0 │ 1 │   ← 모서리 그룹!                            │
│   └───────────────────┘                                        │
│     ↑               ↑                                           │
│   4개 모서리를 하나의 그룹으로                                   │
│                                                                 │
│   그룹 분석:                                                    │
│   - A: 0와 1 모두 포함 → 제거                                   │
│   - B: 0와 1 모두 포함 → 제거                                   │
│   - C: 모두 0                                                   │
│   - D: 0와 1 모두 포함 → 제거                                   │
│   - 결과: F = C'D' (C=0 AND D=0)                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4. Don't Care 조건

```
┌─────────────────────────────────────────────────────────────────┐
│                Don't Care (X) 처리                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Don't Care (X): 출력이 0이든 1이든 상관없는 조건               │
│   - 필요시 1로 처리하여 더 큰 그룹 형성                         │
│   - 불필요시 0으로 처리                                         │
│                                                                 │
│   예시: F = Σm(1, 3) + d(0, 2)                                  │
│                                                                 │
│           BC                                                   │
│       00  01  11  10                                           │
│   ┌───────────────────┐                                        │
│  A│ X │ 1 │ 1 │ X │                                             │
│   ├───┼───┼───┼───┤                                            │
│   │ 0 │ 0 │ 0 │ 0 │                                             │
│   └───────────────────┘                                        │
│                                                                 │
│   X를 1로 처리하면:                                             │
│   - 4개를 하나의 그룹으로                                       │
│   - F = A' (A=0일 때)                                           │
│                                                                 │
│   X를 0으로 처리하면:                                           │
│   - 2개만 그룹                                                  │
│   - F = A'C (더 복잡함)                                         │
│                                                                 │
│   ∴ Don't Care를 활용하면 더 간단한 식 도출!                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **카르노 맵**: 그레이 코드로 배치된 격자를 이용한 논리 함수 최소화
> **그레이 코드**: 인접 코드 간 1비트만 차이

### 2. 빈출 키워드

- **2의 거듭제곱**: 그룹 크기 (1, 2, 4, 8...)
- **순환 경계**: 맵 양 끝 연결
- **Don't Care**: X로 표시, 유리하게 활용
- **최소 곱의 합(SOP)**: 그룹 → 곱항 → 합

### 3. 예상 문제

**Q1. 4변수 카르노 맵에서 모서리 4개가 그룹이 되는 이유를 설명하시오.**

**Q2. F(A,B,C,D) = Σm(0,1,2,3,4,5,6,7)을 카르노 맵으로 최소화하시오.**

**Q3. Don't Care 조건이 카르노 맵 최소화에 미치는 영향을 설명하시오.**
