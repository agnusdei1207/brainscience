# 최소항과 최대항 (Minterm & Maxterm)

## 정의

**최소항(Minterm)**은 진리표에서 **출력이 1인 모든 입력 조합 중에서 1이 포함된 최소항이다 **최대항(Maxterm)**은 진리표에서 **출력이 0인 모든 입력 조합 중에서 0이 포함된다. 부울 함수의 표준 형태(SOP, 또는 POS(Product of Sum) 형태로 표현한다.

```
┌─────────────────────────────────────────────────────────────────┐
│                   최소항과 최대항                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   최소항(Minterm): m = A'·B'·C'                                   │
│                    (AB'C' 포함)                                    │
│                                                                 │
│   예: 3변수(A, B, C)의 최소항                                │
│   m₀ = A'B'C'    (A=0, B=0, C=1)                              │
│   m₁ = A'BC      (A=0, B=0, C=1) → B, C 고정               │
│   m₂ = AB'C       (A=0, B=1, C=0) → C 고정               │
│   m₃ = AB'C       (A=0, B=1, C=1)                              │
│   m₄ = ABC'        (A=1, B=0, C=0)                              │
│   m₅ = ABC'        (A=1, B=0, C=1)                              │
│   m₆ = ABC'        (A=1, B=1, C=0)                              │
│   m₇ = ABC'        (A=1, B=1, C=1)                              │
│                                                                 │
│   최대항(Maxterm): M = A + B + C                                   │
│                    (ABC 합집)                                     │
│                                                                 │
│   예: 3변수(A, B, C)의 최대항                                │
│   M₀ = A + B + C   (A=0, B=0, C=0)                              │
│   M₁ = A + B + C'   (A=0, B=0, C=1)                              │
│   M₂ = A + B + C'   (A=0, B=1, C=0)                              │
│   M₃ = A + B + C    (A=0, B=1, C=1)                              │
│   M₄ = A + B + C    (A=1, B=0, C=0) → A, B 포함               │
│   M₅ = A + B + C'   (A=1, B=0, C=1) → A, C 포함               │
│   M₆ = A + B + C'   (A=1, B=1, C=0) → B 포함                 │
│   M₇ = A + B + C'   (A=1, B=1, C=1)                              │
│                                                                 │
│   💡 기억: 최소항은 곱의 합(SOP), 최대항은 곱의 합(POS) 형태              │
│      SOP: F = Σm(1, 2, 3, ...)                               │
│      POS: F = ΠM(0, 2, 4, ...) = M(0) 포함 안 됼 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 최소항과 최대항의 관계

```
┌─────────────────────────────────────────────────────────────────┐
│           최소항 ↔ 최대항 관계                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   F(최소항) = Σm (곱의 합)                                │
│   F(최대항) = ΠM (곱의 곱)                                │
│                                                                 │
│   💡 관계: F = F' = F'' = F' 성립                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

```

### 2. SOP와 POS 형 부울 대수식 변환

```
┌─────────────────────────────────────────────────────────────────┐
│           SOP vs POS 논리식 표현                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   SOP (곱의 합):                                             │
│   F(A, B, C) = m₀ + m₁ + m₂ + m₃ + m₄ + m₅ + m₆ + m₇             │
│                                                                 │
│   POS (곱의 곱):                                             │
│   F(A, B, C) = M₀·M₁·M₂·M₃·M₄·M₅·M₆·M₇             │
│                                                                 │
│   💡 변환:                                             │
│   F = A'B'C' + A'B'C' + A'B'C  (최소항)                     │
│   F = (A+B+C)(A+B+C)' · (A+B+C)' (최대항)                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **최소항**: 출력이 1인 입력 조합 (AND 형식)
> **최대항**: 출력이 0인 입력 조합 (OR 형식)
> **SOP**: 곱의 합 형태 (AND 게이트 구현)
> **POS**: 곱의 곱 형태 (OR 게이트 구현)

### 2. 빈출 키워드

- **최소항(m)**: m₀, m₁, m₂, ...
- **최대항(M)**: M₀, M₁, M₂, ...
- **SOP, POS**: 논리식 최소화 방법
- **곱의 합, 최소항**: AND 게이트로 구현
- **곱의 곱, 최대항**: OR 게이트로 구현

### 3. 예상 문제
