# 논리 게이트 (Logic Gates)

## 정의

**논리 게이트(Logic Gate)**는 디지털 논리 회로의 기본 구성 요소로, 하나 이상의 입력 신호에 대해 **정해진 하나의 출력 신호**를 생성한다. AND, OR, NOT을 기본 게이트로 시작하여 NAND, NOR, XOR, XNOR 등의 파생 게이트를 포함한다.

```
┌─────────────────────────────────────────────────────────────────┐
│           기본 논리 게이트 심볼                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   AND 게이트:                                                 │
│   ┌────────────────────────────────────────────────────────┐  │
│   │  A  │  B  │  A · B  │                               │  │
│   ├───┼───┼───────┤                               │  │
│   │  0  │  0  │   0   │  ← 모두 1이어야 1              │  │
│   │  0  │  1  │   0   │                               │  │
│   │  1  │  0  │   0   │                               │  │
│   │  1  │  1  │   1   │                               │  │
│   └────────────────────────────────────────────────────────┘  │
│                                                                 │
│   OR 게이트:                                                  │
│   ┌────────────────────────────────────────────────────────┐  │
│   │  A  │  B  │  A + B  │                               │  │
│   ├───┼───┼───────┤                               │  │
│   │  0  │  0  │   0   │  ← 하나라도 1이면 1            │  │
│   │  0  │  1  │   1   │                               │  │
│   │  1  │  0  │   1   │                               │  │
│   │  1  │  1  │   1   │                               │  │
│   └────────────────────────────────────────────────────────┘  │
│                                                                 │
│   NOT 게이트:                                                 │
│   ┌─────────────────┐                                          │
│   │  A  │   Ā   │                                          │
│   ├───┼───────┤                                          │
│   │  0  │   1   │                                          │
│   │  1  │   0   │                                          │
│   └─────────────────┘                                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 기본 게이트 상세

```
┌─────────────────────────────────────────────────────────────────┐
│           기본 게이트 기호 및 표현                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   AND 게이트 (논리곱):                                          │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   기호: A · B, AB, A ∧ B                               │ │
│   │   회로:                                                │ │
│   │        ┌───┐                                           │ │
│   │   A ──┤   │─── Y                                         │ │
│   │        │   │                                             │ │
│   │   B ──┤   │                                             │ │
│   │        └───┘                                           │ │
│   │                                                         │ │
│   │   특징: 모든 입력이 1일 때만 출력 1                            │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   OR 게이트 (논리합):                                           │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   기호: A + B, A ∨ B                               │ │
│   │   회로:                                                │ │
│   │        ┌───┐                                           │ │
│   │   A ──┤   │                                               │ │
│   │        │   │─── Y                                         │ │
│   │   B ──┤   │                                               │ │
│   │        └───┘                                           │ │
│   │                                                         │ │
│   │   특징: 하나라도 1이면 출력 1                              │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   NOT 게이트 (부정):                                           │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   기호: Ā, A', ¬A, ~A                              │ │
│   │   회로:                                                │ │
│   │        ┌───┐                                           │ │
│   │        │   │                                               │ │
│   │   A ──┤   ├─── Y                                         │ │
│   │        │   │                                             │ │
│   │        └───┘                                           │ │
│   │                                                         │ │
│   │   특징: 입력을 반전시킴                              │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 파생 게이트

```
┌─────────────────────────────────────────────────────────────────┐
│           파생 게이트 (Derived Gates)                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   NAND 게이트 (NOT-AND):                                        │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   기호: (A · B)' = Ā · Ḃ                             │ │
│   │   회로: AND + NOT (결합)                              │ │
│   │                                                         │ │
│   │        ┌───┐                                           │ │
│   │   A ──┤   │                                               │ │
│   │        │   │───┐                                           │ │
│   │   B ──┤   │  o─── Y                                │ │
│   │        └───┘                                           │ │
│   │                                                         │ │
│   │   특징: 모든 입력이 1일 때만 출력 0 (AND의 반대)            │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   NOR 게이트 (NOT-OR):                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   기호: (A + B)' = Ā + Ḃ                             │ │
│   │   회로: OR + NOT (결합)                              │ │
│   │                                                         │ │
│   │        ┌───┐                                           │ │
│   │   A ──┤   │                                               │ │
│   │        │   │                                               │ │
│   │   B ──┤   │  o─── Y                                │ │
│   │        └───┘                                           │ │
│   │                                                         │ │
│   │   특징: 모든 입력이 0일 때만 출력 1 (OR의 반대)             │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   XOR 게이트 (배타적 OR):                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   기호: A ⊕ B, A ⊻ B                              │ │
│   │   회로: NOT + OR + NOT                                │ │
│   │                                                         │ │
│   │   진리표:                                              │ │
│   │   ┌───────────────────────────────────────┐ │
│   │   │  A  │  B  │  A ⊕ B  │  A ⊕ B │                   │ │
│   │   ├───┼───┼───────┼───────┤                   │ │
│   │   │  0  │  0  │   0   │   0   │                   │ │
│   │   │  0  │  1  │   1   │   0   │  ← 다르 때 0             │ │
│   │   │  1  │  0  │   1   │   0   │                   │ │
│   │   │  1  │  1  │   0   │   1   │                   │ │
│   │   └───────────────────────────────────────┘ │
│   │                                                         │ │
│   │   특징: 입력이 다르 때 1 (홀수 1)                          │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   XNOR 게이트 (배타적 NOR):                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   기호: A ⊙ B, A ⊙ B                              │ │
│   │                                                         │ │
│   │   진리표:                                              │ │
│   │   ┌───────────────────────────────────────┐ │
│   │   │  A  │  B  │  A ⊙ B  │  A ⊙ B │                   │ │
│   │   ├───┼───┼───────┼───────┤                   │ │
│   │   │  0  │  0  │   1   │   1   │  ← 같을 때 1             │ │
│   │   │  0  │  1  │   0   │   0   │  ← 다를 때 0             │ │
│   │   │  1  │  0  │   0   │   0   │                   │ │
│   │   │  1  │  1  │   1   │   0   │                   │ │
│   │   └───────────────────────────────────────┘ │
│   │                                                         │ │
│   │   특징: 입력이 같을 때 0 (동수 0)                         │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **AND 게이트**: 모든 입력이 1일 때만 출력 1
> **OR 게이트**: 하나라도 1이면 출력 1
> **NOT 게이트**: 입력을 반전
> **NAND**: AND의 반대, 모든 입력이 1일 때만 0
> **NOR**: OR의 반대, 모든 입력이 0일 때만 1

### 2. 빈출 키워드

- **7대 기본 게이트**: AND, OR, NOT
- **NAND, NOR**: 범용 게이트 (모든 게이트 구현 가능)
- **XOR**: 배타적 OR, 입력 다름 시 0
- **XNOR**: 배타적 NOR, 입력 같을 때 0
