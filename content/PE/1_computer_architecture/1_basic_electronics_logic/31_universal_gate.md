# 범용 게이트 (Universal Gate)

## 정의

**범용 게이트(Universal Gate)**는 **단독으로 모든 논리 함수를 구현**할 수 있는 게이트다. NAND 게이트와 NOR 게이트가 이에 해당하며, 이 두 게이트만으로 AND, OR, NOT 게이트를 모두 구성할 수 있다.

## 핵심 개념

### 1. NAND로 기본 게이트 구현

```
┌─────────────────────────────────────────────────────────────────┐
│           NAND 게이트로 기본 게이트 구현                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   NOT 게이트 구현:                                               │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   논리식: Y = (A · A)' = A'                              │ │
│   │                                                         │ │
│   │       A ──┬──┤  NAND  ├── Y = Ā                         │ │
│   │           │                                             │ │
│   │           └──┤                                             │ │
│   │                                                         │ │
│   │   설명: NAND의 두 입력을 같게 하면 NOT가 됨               │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   AND 게이트 구현:                                               │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   논리식: Y = ((A · B)')' = A · B                        │ │
│   │                                                         │ │
│   │       A ──┐                                              │ │
│   │           │ NAND ──┬──┤  NAND  ├── Y = A·B              │ │
│   │       B ──┘        │                                      │ │
│   │                    └──┤                                    │ │
│   │                                                         │ │
│   │   설명: NAND 출력을 다시 NAND(NOT)로 반전                 │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   OR 게이트 구현:                                                │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   논리식: Y = (A' · B')' = A + B  (드모르간)              │ │
│   │                                                         │ │
│   │       A ──┬──┤  NAND  ──┐                              │ │
│   │           │              │                              │ │
│   │           └──┤          ├──┤  NAND  ├── Y = A+B         │ │
│   │                          │                              │ │
│   │       B ──┬──┤  NAND  ──┘                              │ │
│   │           │                                             │ │
│   │           └──┤                                             │ │
│   │                                                         │ │
│   │   설명: 각 입력을 NOT(NAND)한 후 NAND로 결합              │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 왜 NAND가 산업 표준인가

```
┌─────────────────────────────────────────────────────────────────┐
│           NAND 게이트가 선호되는 이유                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   CMOS 관점:                                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   NAND 게이트:                                           │ │
│   │   • PMOS 2개 병렬 + NMOS 2개 직렬                        │ │
│   │   • 트랜지스터 4개                                       │ │
│   │   • 빠른 스위칭 (PMOS가 병렬)                             │ │
│   │                                                         │ │
│   │   NOR 게이트:                                            │ │
│   │   • PMOS 2개 직렬 + NMOS 2개 병렬                        │ │
│   │   • 트랜지스터 4개                                       │ │
│   │   • 느린 스위칭 (PMOS가 직렬)                             │ │
│   │                                                         │ │
│   │   💡 PMOS는 NMOS보다 느림 (정공 이동도 < 전자 이동도)      │ │
│   │   💡 NAND에서 PMOS가 병렬 → 더 빠름                       │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   성능 비교:                                                     │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  구분         │  NAND 게이트    │  NOR 게이트            │ │
│   ├───────────────┼─────────────────┼───────────────────────┤ │
│   │ 전파 지연    │  낮음           │  높음                 │ │
│   │ (Delay)      │  (Fast)         │  (Slow)               │ │
│   ├───────────────┼─────────────────┼───────────────────────┤ │
│   │ 전력 소모    │  낮음           │  높음                 │ │
│   │ (Power)      │                 │                       │ │
│   ├───────────────┼─────────────────┼───────────────────────┤ │
│   │ 칩 면적      │  작음           │  비슷함               │ │
│   │ (Area)       │                 │                       │ │
│   ├───────────────┼─────────────────┼───────────────────────┤ │
│   │ 산업 선호도  │  높음           │  낮음                 │ │
│   └───────────────┴─────────────────┴───────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **범용 게이트**: 단독으로 모든 논리 함수를 구현할 수 있는 게이트
> **NAND 게이트**: 산업 표준 범용 게이트, CMOS에서 효율적
> **NOR 게이트**: 범용 게이트지만 NAND보다 느림

### 2. 빈출 키워드

- **범용성**: NAND/NOR만으로 모든 논리 회로 구현
- **CMOS 효율**: NAND가 NOR보다 빠름 (PMOS 병렬)
- **드모르간 법칙**: NAND ↔ OR의 부정, NOR ↔ AND의 부정
- **산업 표준**: NAND 게이트 선호

### 3. 예상 문제

**Q1. 범용 게이트의 정의를 내리고 NAND와 NOR 게이트가 범용 게이트인 이유를 설명하시오.**

**Q2. NAND 게이트만을 사용하여 NOT, AND, OR 게이트를 구현하시오.**

**Q3. 산업 현장에서 NAND 게이트가 NOR 게이트보다 선호되는 이유를 CMOS 관점에서 설명하시오.**
