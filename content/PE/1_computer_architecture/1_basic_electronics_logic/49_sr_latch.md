# 49. SR 래치 (Set-Reset Latch)

## Ⅰ. 개요

SR 래치(Set-Reset Latch)는 디지털 논리 회로에서 가장 단순하고 원초적인 형태의 기억 소자(Memory Element)로, 두 개의 입력 신호인 Set(S)과 Reset(R)을 통해 단일 비트의 데이터를 저장하고 유지하는 비동기식 순차 논리 회로입니다. Set 신호를 가하면 출력이 '1'로 상태가 변하고(저장되고), Reset 신호를 가하면 출력이 '0'으로 초기화됩니다. 두 입력이 모두 비활성 상태일 때는 직전의 출력 상태를 그대로 유지(기억)하는 특성을 갖습니다. SR 래치는 단독으로 사용되기보다는 복잡한 플립플롭이나 동기식 래치를 구성하기 위한 근본적인 기저 회로(Base Circuit)로서 컴퓨터 아키텍처의 상태 유지 메커니즘을 이해하는 핵심 출발점입니다.

## Ⅱ. 아키텍처 및 핵심 원리

SR 래치는 주로 논리 게이트의 출력이 다른 게이트의 입력으로 되돌아가는 교차 궤환(Cross-Coupled) 피드백 루프 구조로 설계됩니다. 구현에 사용되는 게이트 종류에 따라 NOR 기반 래치와 NAND 기반 래치로 나뉩니다.

### 1. NOR 게이트 기반 SR 래치 아키텍처 (Active-High)

```text
[ NOR 게이트 교차 결합 SR 래치 아키텍처 ]

        Set (S) -------------+
                             |
                             V
                           +---\
                           |NOR \
                       +---| G1  |------+-----> Q (Output)
                       |   |    /       |
                       |   +---/        |
                       |      o         |
                       |      |         |
                       +----------------+
                              |         |  <-- Cross-coupled Feedback
                       +----------------+
                       |      |
                       |      o
                       |   +---\
                       |   |NOR \
                       +---| G2  |------+-----> Q' (Inverted Output)
                           |    /
        Reset (R) ---------+---/
```

*   **동작 원리 및 진리표 (NOR 기반)**
    *   **S=0, R=0 (유지 모드)**: 외부 입력이 없으므로, 피드백 루프에 의해 이전 출력 상태(Q, Q')가 그대로 유지됩니다.
    *   **S=1, R=0 (Set 모드)**: S가 1이 되면 NOR 게이트 특성상 G1의 출력 Q는 무조건 0이 됩니다. 이 Q=0이 G2의 입력으로 들어가 R=0과 만나면 G2의 출력 Q'는 1이 됩니다. 결과적으로 상태를 Set 시키지만, 이 회로에서 Q'가 주 출력이 되는 형태로 설계되기도 합니다. 일반적으로 정규화된 S, R 입력 회로에서는 S가 Q를 1로 만듭니다. (*주의: 위 다이어그램 배선 연결 방식에 따라 Q와 Q'의 위치가 반전될 수 있습니다. 표준적으로 S가 입력되는 NOR 게이트의 출력이 Q'입니다.*)
    *   **S=0, R=1 (Reset 모드)**: R이 1이 되면 Q'=0이 되고 이를 피드백받아 Q=1이 됩니다. 상태가 초기화됩니다.
    *   **S=1, R=1 (금지된 상태 / Invalid State)**: S와 R이 모두 1이면 두 NOR 게이트의 출력이 모두 0이 되어 Q=0, Q'=0이 발생합니다. 이는 Q와 Q'가 서로 반전된 값이어야 한다는 논리적 기본 전제에 위배되며, S와 R이 동시에 0으로 떨어질 경우 피드백 루프에 의해 레이스 컨디션이 발생하여 다음 상태를 예측할 수 없는 불안정한 진동(메타스테빌리티) 상태에 빠지게 되므로 절대 사용해서는 안 되는 금지 구역입니다.

### 2. NAND 게이트 기반 S'R' 래치 아키텍처 (Active-Low)

NAND 게이트로 구성된 래치는 입력이 0일 때 활성화되는 Active-Low 특성을 가지므로 통상 S'R' 래치라고 부릅니다.

```text
[ NAND 게이트 교차 결합 S'R' 래치 아키텍처 ]

        Set' (S') -----------+
                             |
                             V
                           +---\
                           |NAND\
                       +---| G1  |o-----+-----> Q (Output)
                       |   |    /       |
                       |   +---/        |
                       |                |
                       |                |
                       +----------------+
                              |         |  <-- Cross-coupled Feedback
                       +----------------+
                       |      |
                       |      |
                       |   +---\
                       |   |NAND\
                       +---| G2  |o-----+-----> Q' (Inverted Output)
                           |    /
        Reset' (R') -------+---/
```

## Ⅲ. 융합 비교 및 다각도 분석

### 1. NOR 래치와 NAND 래치의 논리적 쌍대성(Duality)
디지털 회로 설계의 쌍대성 원리에 따라 NOR 래치와 NAND 래치는 정확히 대칭적인 동작 특성을 보입니다. NOR 래치는 S, R이 '1'일 때 동작하는 반면, NAND 래치는 S', R'가 '0'일 때 동작합니다. 또한 유지 상태를 위한 입력 조건도 NOR는 (0,0), NAND는 (1,1)로 상호 보완적입니다. 최신 CMOS 반도체 공정에서는 P-MOS 트랜지스터의 직렬연결이 필요한 NOR 게이트보다 트랜지스터 배치와 동작 속도 면에서 유리한 NAND 게이트를 기반으로 메모리 셀을 구현하는 것이 훨씬 일반적이고 효율적인 선택입니다.

### 2. 한계점 분석: 상태 불확실성과 동기화 문제
SR 래치의 가장 큰 약점은 두 입력이 동시에 활성화되었을 때 발생하는 '금지된 상태'입니다. 이 상태로의 진입 자체도 문제지만, 빠져나올 때 어느 출력이 먼저 회복될지 알 수 없어 시스템 전체의 비결정적 동작을 유발합니다. 또한 클럭 신호에 의한 동기화 메커니즘이 없기 때문에, 입력 신호에 포함된 미세한 노이즈나 글리치가 여과 없이 곧바로 출력 상태를 영구적으로 뒤집어 버릴 수 있는 치명적인 노이즈 취약성을 지니고 있습니다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 1. 스위치 디바운싱(Switch Debouncing) 하드웨어 회로
SR 래치가 실무에서 단독으로 그 진가를 발휘하는 대표적인 응용 분야가 기계적 스위치의 디바운싱 회로입니다.
사용자가 기계식 푸시 버튼을 누를 때, 금속 접점이 닿는 순간의 탄성으로 인해 수 밀리초(ms) 동안 수십 번 접점이 붙었다 떨어지기를 반복하는 바운싱(Bouncing) 현상이 발생합니다. 논리 회로는 이를 수십 번의 신호 입력으로 오인할 수 있습니다.
*   SPDT(Single Pole Double Throw) 스위치의 양 끝단을 S'R' 래치의 입력에 연결하면, 스위치가 처음 닿아 첫 번째 논리 '0' 신호가 래치에 인가되는 순간 상태가 변하여 래치업(Latch-up) 됩니다.
*   그 직후 발생하는 바운싱에 의한 순간적인 개방 상태(입력이 1,1인 유지 상태)는 래치의 이전 상태 기억 특성 덕분에 완전히 무시됩니다. 소프트웨어 딜레이를 쓰지 않고 하드웨어적으로 완벽하게 깨끗하고 단일한 구형파 에지를 만들어내는 매우 견고하고 우아한 기술적 해결책입니다.

### 2. 기술사적 판단 관점
디지털 시스템 설계 시 SR 래치는 원시적이기 때문에 그 자체로 메인 데이터 버스나 레지스터 파일에 사용되어서는 안 됩니다. 기술사는 반드시 SR 래치 앞단에 게이트 논리를 추가하여 S와 R이 동시에 활성화되는 조건을 원천적으로 차단하는 설계(예: D 래치 아키텍처로의 전환)를 강제해야 하며, 노이즈에 노출될 수 있는 비동기 입력 인터페이스에 한정하여 엄격한 검토 하에 제한적으로 적용해야 합니다.

## Ⅴ. 기대효과 및 결론

SR 래치는 정보 기억이라는 추상적인 개념을 물리적인 전기 회로의 피드백 루프로 구현해낸 디지털 컴퓨터 역사상 가장 위대한 발명 중 하나입니다. S=1, R=1이라는 불완전하고 금지된 상태의 존재는 결함이라기보다는, 복잡도를 최소화하여 가장 빠르고 직관적인 상태 제어를 달성하기 위한 구조적 트레이드오프의 결과물로 보아야 합니다. 
결론적으로 SR 래치는 그 자체로서의 활용 빈도는 낮아졌을지라도, 오늘날 최고 성능을 자랑하는 모든 마이크로프로세서 내부의 수억 개 플립플롭과 SRAM 셀의 가장 깊숙한 곳에서 여전히 데이터를 꽉 움켜쥐고 있는 심장과도 같은 역할을 수행하고 있습니다.
