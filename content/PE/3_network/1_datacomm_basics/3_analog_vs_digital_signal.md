# 아날로그 신호 vs 디지털 신호

## 정의

**아날로그 신호**는 시간에 따라 **연속적으로 변화**하는 물리량(전압, 전류 등)이며, **디지털 신호**는 **이산적인 값(0과 1)**만을 취하는 신호다.

```
┌─────────────────────────────────────────────────────────────────┐
│           아날로그 신호 vs 디지털 신호 파형                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   아날로그 신호:                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  ▲                                                       │ │
│   │  │      ╭──╮                                             │ │
│   │  │    ╭╯    ╰╮   ╭╮                                     │ │
│   │  │ ╭╮╭╯      ╰╮╭╯ ╰╮  ╭╮                                │ │
│   │  │╭╯╰╯         ╰╯   ╰╮╭╯ ╰╮                              │ │
│   │  ╰╯                  ╰╯   ╰──▶ 시간                       │ │
│   │                                                         │ │
│   │  특징: 연속적, 무한한 값, 자연스러운 파형                 │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   디지털 신호:                                                   │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  ▲                                                       │ │
│   │  │    ┌──┐      ┌──┐  ┌──┐                              │ │
│   │  │    │  │      │  │  │  │     1                        │ │
│   │  │ ┌──┤  ├──┬───┤  ├──┤  ├──┬───                        │ │
│   │  │ │  │  │  │  │  │  │  │  │                            │ │
│   │  │─┤  │  │  └──┤  │  └──┤  └──┤──▶ 시간                  │ │
│   │  │ │            │          │    0                        │ │
│   │  └──┴──────────┴──────────┴───                           │ │
│   │                                                         │ │
│   │  특징: 이산적, 유한한 값(0/1), 직사각형 파형              │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 신호 특성 비교

```
┌─────────────────────────────────────────────────────────────────┐
│           아날로그 vs 디지털 신호 특성 비교                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  구분       │  아날로그 신호     │  디지털 신호           │ │
│   ├─────────────┼────────────────────┼───────────────────────┤ │
│   │ 값의 특성   │  연속적, 무한      │  이산적, 유한(0/1)     │ │
│   ├─────────────┼────────────────────┼───────────────────────┤ │
│   │ 파형        │  부드러운 곡선      │  직사각형(계단)        │ │
│   ├─────────────┼────────────────────┼───────────────────────┤ │
│   │ 잡음 영향   │  누적됨            │  재생 가능            │ │
│   ├─────────────┼────────────────────┼───────────────────────┤ │
│   │ 저장/복제   │  품질 저하         │  완벽한 복제           │ │
│   ├─────────────┼────────────────────┼───────────────────────┤ │
│   │ 대역폭      │  상대적 적음       │  더 많이 필요          │ │
│   ├─────────────┼────────────────────┼───────────────────────┤ │
│   │ 장비        │  증폭기(Amplifier) │  리피터(Repeater)     │ │
│   ├─────────────┼────────────────────┼───────────────────────┤ │
│   │ 처리        │  하드웨어 집약적    │  소프트웨어 처리 가능  │ │
│   └─────────────┴────────────────────┴───────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 잡음(Noise) 영향

```
┌─────────────────────────────────────────────────────────────────┐
│               잡음이 신호에 미치는 영향                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   아날로그 신호 - 잡음 누적:                                     │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   원본:    ∿∿∿∿  (깨끗한 파형)                          │ │
│   │                                                         │ │
│   │   전송 후: ∿⌇∿⌇∿  (잡음 섞임)                           │ │
│   │            ↖ 잡음                                       │ │
│   │                                                         │ │
│   │   증폭:    ⌇⌇⌇⌇⌇⌇  (잡음까지 증폭!)                    │ │
│   │                                                         │ │
│   │   → 잡음이 계속 누적되어 품질 저하                       │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   디지털 신호 - 재생 가능:                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   원본:    ┌──┐  ┌──┐  (깨끗한 펄스)                    │ │
│   │                                                         │ │
│   │   전송 후: ┌╥╥┐  ┌╥╥┐  (잡음 섞임)                      │ │
│   │            ↖ 잡음                                       │ │
│   │                                                         │ │
│   │   재생:    ┌──┐  ┌──┐  (완벽한 복원!)                   │ │
│   │            ↖ 리피터                                     │ │
│   │                                                         │ │
│   │   → 임계값 이상이면 1, 아니면 0으로 복원                  │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 전송 효율 비교

```
┌─────────────────────────────────────────────────────────────────┐
│           아날로그 vs 디지털 통신 효율                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   아날로그 통신:                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   [음성] ──▶ [변조] ──▶ [전송] ──▶ [복조] ──▶ [음성]    │ │
│   │             AM/FM                                        │ │
│   │                                                         │ │
│   │   장점:                                                  │ │
│   │   • 대역폭 효율적 (단일 채널로 전송)                      │ │
│   │   • 단순한 하드웨어                                      │ │
│   │   • 실시간 전송에 유리                                   │ │
│   │                                                         │ │
│   │   단점:                                                  │ │
│   │   • 잡음 취약, 품질 저하                                 │ │
│   │   • 보안 취약 (도청 쉬움)                                │ │
│   │   • 다중화 제한                                          │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   디지털 통신:                                                   │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   [데이터]─▶[부호화]─▶[변조]─▶[전송]─▶[복조]─▶[복호]─▶   │ │
│   │            PCM      QAM                                │ │
│   │                                                         │ │
│   │   장점:                                                  │ │
│   │   • 잡음 내성 (재생 가능)                                │ │
│   │   • 암호화 가능 (보안)                                   │ │
│   │   • 다중화 용이 (TDM)                                    │ │
│   │   • 오류 검출/정정 가능                                  │ │
│   │                                                         │ │
│   │   단점:                                                  │ │
│   │   • 대역폭 요구 증가                                     │ │
│   │   • 동기화 필요                                          │ │
│   │   • A/D 변환 오버헤드                                    │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4. 장비 비교

```
┌─────────────────────────────────────────────────────────────────┐
│           증폭기(Amplifier) vs 리피터(Repeater)                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   증폭기 (아날로그):                                             │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   약한 신호 ──▶ [증폭기] ──▶ 강한 신호                   │ │
│   │                 (모든 것 증폭)                           │ │
│   │                                                         │ │
│   │   • 신호 + 잡음 모두 증폭                                │ │
│   │   • SNR(신호대잡음비) 개선 없음                          │ │
│   │   • 아날로그 신호 거리 연장                              │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   리피터 (디지털):                                               │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   약한 신호 ──▶ [복원] ──▶ [재생] ──▶ 깨끗한 신호        │ │
│   │                 (0/1 판단)                               │ │
│   │                                                         │ │
│   │   • 잡음 제거 후 신호만 재생                             │ │
│   │   • SNR 완전 복원                                       │ │
│   │   • 디지털 신호 거리 연장                                │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **아날로그 신호**: 연속적, 무한한 값
> **디지털 신호**: 이산적, 0과 1만 사용

### 2. 빈출 키워드

- **재생성(Regeneration)**: 디지털 신호의 잡음 제거 특성
- **증폭기 vs 리피터**: 아날로그/디지털 신호 연장 장비
- **SNR**: 신호대잡음비
- **변조/복조**: 신호 전송을 위한 변환

### 3. 예상 문제

**Q1. 디지털 신호가 아날로그 신호보다 잡음에 강한 이유를 설명하시오.**

**Q2. 증폭기와 리피터의 차이점을 기술하시오.**

**Q3. 디지털 통신의 장단점을 설명하시오.**
