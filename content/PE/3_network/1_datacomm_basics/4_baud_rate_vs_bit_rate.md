# 보드레이트 vs 비트레이트

## 정의

**보드레이트(Baud Rate)**는 **초당 신호 변화(심볼) 횟수**이며, **비트레이트(Bit Rate)**는 **초당 전송되는 비트 수**다. 두 개념은 밀접히 관련되지만 동일하지 않다.

```
┌─────────────────────────────────────────────────────────────────┐
│           보드레이트 vs 비트레이트 개념도                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   비트레이트 (Bit Rate):                                        │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   단위: bps (bits per second)                           │ │
│   │                                                         │ │
│   │   시간 ──────────────────────────────────────────────▶  │ │
│   │       │ 1 │ 0 │ 1 │ 1 │ 0 │ 0 │ 1 │ 0 │               │ │
│   │       └───┴───┴───┴───┴───┴───┴───┴───┘               │ │
│   │                                                     ↑   │ │
│   │                                          초당 8비트 전송   │ │
│   │                                          = 8 bps         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   보드레이트 (Baud Rate):                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   단위: Bd (Baud) = 심볼/초                              │ │
│   │                                                         │ │
│   │   시간 ──────────────────────────────────────────────▶  │ │
│   │       │  S1 │  S2 │  S3 │  S4 │  S5 │                  │ │
│   │       └────┴────┴────┴────┴────┘                       │ │
│   │                                          ↑               │ │
│   │                                 초당 5회 신호 변화         │ │
│   │                                 = 5 Bd                   │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 심볼(Symbol)과 비트의 관계

```
┌─────────────────────────────────────────────────────────────────┐
│           심볼당 비트 수에 따른 비트레이트                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   핵심 공식:                                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   비트레이트 = 보드레이트 × log₂(M)                      │ │
│   │                                                         │ │
│   │   M = 심볼당 가능한 상태 수 (신호 레벨 수)               │ │
│   │                                                         │ │
│   │   비트레이트 = 보드레이트 × n                            │ │
│   │   (n = 심볼당 비트 수 = log₂(M))                         │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   2진 신호 (M=2, 1심볼=1비트):                                  │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   신호 레벨: 0, 1 (2개)                                  │ │
│   │   심볼당 비트: 1                                         │ │
│   │   비트레이트 = 보드레이트                                │ │
│   │                                                         │ │
│   │   예: 1200 Bd = 1200 bps                                │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   4진 신호 (M=4, 1심볼=2비트):                                  │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   신호 레벨: 00, 01, 10, 11 (4개)                        │ │
│   │   심볼당 비트: 2                                         │ │
│   │   비트레이트 = 2 × 보드레이트                            │ │
│   │                                                         │ │
│   │   예: 1200 Bd = 2400 bps                                │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────────┐
│           다양한 변조 방식의 비교                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   2진 신호 (1심볼 = 1비트):                                     │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   ┌──┐    ┌──┐                                          │ │
│   │   │  │    │  │    1                                     │ │
│   │───┤  ├───┤  ├───                                        │ │
│   │   │            0                                        │ │
│   │                                                         │ │
│   │   보드레이트 = 비트레이트                                │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   4진 신호 (1심볼 = 2비트):                                     │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   ┌────┐         ┌────┐                                 │ │
│   │   │ +3 │         │ +3 │     11                         │ │
│   │───┤    ├───┬─────┤    ├───                             │ │
│   │   │ +1 │   │     │ +1 │     10                         │ │
│   │───┤    ├───┴─────┤    ├───                             │ │
│   │   │ -1 │         │ -1 │     01                         │ │
│   │───┤    ├─────────┤    ├───                             │ │
│   │   │ -3 │         │ -3 │     00                         │ │
│   │                                                         │ │
│   │   비트레이트 = 2 × 보드레이트                            │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   QPSK 예시:                                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   보드레이트: 1200 Bd                                    │ │
│   │   심볼당 비트: 2 (4가지 위상)                             │ │
│   │   비트레이트: 1200 × 2 = 2400 bps                        │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 전송 속도 vs 변조 속도

```
┌─────────────────────────────────────────────────────────────────┐
│       전송 속도(Bit Rate) vs 변조 속도(Baud Rate)                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  구분      │  전송 속도       │  변조 속도               │ │
│   │            │  (Bit Rate)     │  (Baud Rate)             │ │
│   ├────────────┼─────────────────┼─────────────────────────┤ │
│   │ 정의       │ 초당 비트 수     │ 초당 신호 변화 횟수       │ │
│   ├────────────┼─────────────────┼─────────────────────────┤ │
│   │ 단위       │ bps (bit/s)     │ Bd (baud)               │ │
│   ├────────────┼─────────────────┼─────────────────────────┤ │
│   │ 의미       │ 정보 전송량      │ 신호 변조 횟수           │ │
│   ├────────────┼─────────────────┼─────────────────────────┤ │
│   │ 관계       │ ≥ 보드레이트    │ ≤ 비트레이트             │ │
│   │            │                 │ (2진 변조 시에만 같음)   │ │
│   ├────────────┼─────────────────┼─────────────────────────┤ │
│   │ 결정 요소  │ 대역폭, 변조방식 │ 대역폭                   │ │
│   └────────────┴─────────────────┴─────────────────────────┘ │
│                                                                 │
│   니키스트 한계:                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   최대 보드레이트 = 2 × 대역폭                           │ │
│   │                                                         │ │
│   │   예: 대역폭 3000Hz → 최대 6000 Bd                      │ │
│   │                                                         │ │
│   │   비트레이트를 높이려면:                                 │ │
│   │   • 심볼당 비트 수 증가 (다진 변조)                      │ │
│   │   • 대역폭 증가                                          │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4. 비트 시간과 심볼 시간

```
┌─────────────────────────────────────────────────────────────────┐
│           비트 시간 vs 심볼 시간                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   비트 시간 (Bit Duration):                                     │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   Tb = 1 / 비트레이트                                   │ │
│   │                                                         │ │
│   │   예: 1 Mbps → Tb = 1 / 1,000,000 = 1 μs               │ │
│   │                                                         │ │
│   │   한 비트가 전송되는 데 걸리는 시간                      │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   심볼 시간 (Symbol Duration):                                  │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   Ts = 1 / 보드레이트                                   │ │
│   │                                                         │ │
│   │   예: 1200 Bd → Ts = 1 / 1200 ≈ 833 μs                 │ │
│   │                                                         │ │
│   │   한 심볼이 전송되는 데 걸리는 시간                      │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   관계:                                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   Ts = n × Tb    (n = 심볼당 비트 수)                   │ │
│   │                                                         │ │
│   │   2진: Ts = Tb                                          │ │
│   │   4진: Ts = 2Tb                                         │ │
│   │   16진: Ts = 4Tb                                        │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **비트레이트**: 초당 전송 비트 수 (bps)
> **보드레이트**: 초당 신호 변화 횟수 (Bd)
> **공식**: 비트레이트 = 보드레이트 × log₂(M)

### 2. 빈출 키워드

- **심볼**: 한 번의 신호 변화 단위
- **M진 변조**: M개 신호 레벨 사용
- **니키스트 한계**: 최대 보드레이트 = 2 × 대역폭
- **Tb, Ts**: 비트 시간, 심볼 시간

### 3. 예상 문제

**Q1. 보드레이트가 2400 Bd이고 QPSK 변조를 사용할 때 비트레이트를 계산하시오.**

**Q2. 56kbps 모뎀이 2400 Bd로 동작한다면 심볼당 몇 비트인가?**

**Q3. 비트레이트를 높이기 위한 방법을 설명하시오.**
