# 대역폭 (Bandwidth)

## 정의

**대역폭(Bandwidth)**은 통신 채널이 **단위 시간당 전송할 수 있는 데이터의 양**이다. 주파수(Hz)로 표현하거나 비트레이트(bps)로 표현한다.

```
┌─────────────────────────────────────────────────────────────────┐
│           대역폭 개념도                                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   시간 영역:                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  ▲                                                │ │
│   │  │ ════════════════════════════════════════════════▶   │ │
│   │  │                                                │ │
│   │  │   최대 주파수 성분 (fmax)                        │ │
│   │  │       ───────────────────────────                     │ │
│   │  │          │           │         │                     │ │
│   │  │          │  대역폭  │         │                     │ │
│   │  │          │           │         │                     │ │
│   │  │          └──────────────────────────                     │ │
│   │  │                                                │ │
│   │  └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   💡 대역폭 = 최고 주파수 - 최저 주파수                          │
│                                                                 │
│   디지털 영역:                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   비트레이트 (bps)                                      │ │
│   │   │                                                    │ │
│   │   │  ═══════════════════════════════════════════════▶   │ │
│   │   │                                                    │ │
│   │   │   초당 비트 전송량                                  │ │
│   │   │                                                    │ │
│   │   └─────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 대역폭과 데이터 전송률

```
┌─────────────────────────────────────────────────────────────────┐
│           대역폭 vs 데이터 전송률                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  구분        │  대역폭          │  데이터 전송률       │ │
│   ├──────────────┼─────────────────────┼─────────────────────┤ │
│   │ 정의        │ 채널 용량          │ 실제 전송량          │ │
│   │             │ (Capacity)        │ (Actual Throughput)│ │
│   ├──────────────┼─────────────────────┼─────────────────────┤ │
│   │ 단위        │ Hz               │ bps                 │ │
│   ├──────────────┼─────────────────────┼─────────────────────┤ │
│   │ 의미        │ 이론적 최대치        │ 실제 달성치             │ │
│   ├──────────────┼─────────────────────┼─────────────────────┤ │
│   │ 관계        │ 대역폭 ≥ 데이터 전송률                       │ │
│   └──────────────┴─────────────────────┴─────────────────────┘ │
│                                                                 │
│   예시:                                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │ 대역폭: 100 MHz (이론적 최대)                              │ │
│   │ 데이터 전송률: 80 Mbps (실제 달성)                             │ │
│   │                                                         │ │
│   │ 💡 대역폭은 잠재력, 데이터 전송률은 실현              │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 섀논의 정리와 대역폭

```
┌─────────────────────────────────────────────────────────────────┐
│           섀논의 정리와 대역폭 관계                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   섀논의 정리 (Shannon-Hartley Theorem):                             │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   C = B × log₂(1 + S/N)                                │ │
│   │                                                         │ │
│   │   C: 채널 용량 (bps)                                     │ │
│   │   B: 대역폭 (Hz)                                        │ │
│   │   S/N: 신호 대 잡음비 (Signal-to-Noise Ratio)            │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   💡 대역폭이 클수록 채널 용량 증가                            │
│   💡 S/N이 높을수록 채널 용량 증가                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 대역폭 효율

```
┌─────────────────────────────────────────────────────────────────┐
│           대역폭 효율 (Bandwidth Efficiency)                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   정의: 단위 대역폭당 전송되는 비트 수                           │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   대역폭 효율 = 데이터 전송률 / 대역폭                      │ │
│   │                 = R / B                                        │ │
│   │                                                         │ │
│   │   단위: bps/Hz 또는 bit/s/Hz                         │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   변조 방식별 대역폭 효율:                                     │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  변조 방식   │  대역폭 효율 (이론적)      │             │ │
│   ├──────────────┼─────────────────────────────────────┤             │ │
│   │ BPSK        │  1 bit/s/Hz                     │             │ │
│   │ QPSK        │  2 bit/s/Hz                     │             │ │
│   │ 8-PSK       │  3 bit/s/Hz                     │             │ │
│   │ 16-QAM      │  4 bit/s/Hz                     │             │ │
│   │ 64-QAM      │  6 bit/s/Hz                     │             │ │
│   └──────────────┴─────────────────────────────────────┘             │
│                                                                 │
│   💡 변조 기술이 발전할수록 대역폭 효율 향상                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **대역폭**: 통신 채널이 단위 시간당 전송할 수 있는 데이터 양
> **섀논의 정리**: C = B × log₂(1 + S/N)
> **대역폭 효율**: 단위 대역폭당 전송 비트 수

### 2. 빈출 키워드

- **Hz vs bps**: 아날로그 vs 디지털 단위
- **채널 용량**: 이론적 최대 전송량
- **S/N 비**: 신호 대 잡음비
- **대역폭 효율**: 주파수당 비트 수

### 3. 예상 문제
