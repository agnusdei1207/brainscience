# 기업정보시스템 키워드 목록

정보통신기술사·컴퓨터응용시스템기술사 대비 기업정보시스템 전 영역 핵심 키워드

---

## 1. 전사적 정보 시스템 전략 — 20개

1. IT 전략 (IT Strategy) — 비즈니스와 IT 정렬 (Business-IT Alignment)
2. EA (Enterprise Architecture) — 자버먼 프레임워크, TOGAF, DoDAF, FEAF
3. TOGAF (The Open Group Architecture Framework) — ADM 사이클
4. 정보화 전략 계획 (ISP, Information Strategy Planning)
5. BPR (Business Process Reengineering) — 프로세스 근본 재설계
6. BPM (Business Process Management) — BPMN 2.0, 프로세스 최적화
7. BPMN (Business Process Model and Notation) — 표준 프로세스 표기
8. 디지털 전환 (DX) — 비즈니스 모델 혁신
9. IT 거버넌스 — IT 의사결정 구조, COBIT 5/2019
10. COBIT (Control Objectives for IT) — IT 통제 프레임워크
11. ITIL (IT Infrastructure Library) v4 — 서비스 가치 시스템 (SVS)
12. IT BSC (Balanced Scorecard) — 재무/고객/내부프로세스/학습·성장 4관점
13. 벤치마킹 (Benchmarking) — 내부/경쟁/기능/전략적
14. 정보시스템 감리 — 감리 기준, 감리원 자격
15. 정보화 수준 진단 (IMS) — 성숙도 평가
16. OKR (Objectives and Key Results) — 목표와 핵심 결과
17. KPI (Key Performance Indicator) — 성과 측정 지표
18. ROI (Return on Investment) — 정보시스템 투자수익률
19. TCO (Total Cost of Ownership) — 총소유비용
20. IT 아웃소싱 (IT Outsourcing) — BPO, SaaS 전환

---

## 2. ERP (전사적 자원 관리) — 18개

1. ERP (Enterprise Resource Planning) 개념 — 통합 경영 정보 시스템
2. ERP 구성 모듈 — 재무/인사/생산/구매/물류/판매
3. ERP 도입 효과 — 실시간 정보, 프로세스 통합, 업무 표준화
4. ERP 도입 방법 — Big Bang / Phased / Pilot
5. SAP ERP / SAP S/4HANA — HANA 인메모리 DB
6. Oracle ERP (Fusion) — 클라우드 ERP
7. 국산 ERP — 더존 iCUBE, 영림원 K-System
8. Cloud ERP — SaaS 기반, Microsoft Dynamics 365
9. ERP 커스터마이징 vs 표준화 — 갭 분석 (Gap Analysis)
10. ERP 변경 관리 (Change Management)
11. 핵심 성과 지표 (KPI) — ERP 도입 후 측정
12. ERP 재무회계 — 원장, 채권/채무, 결산
13. ERP 관리회계 — 원가 관리, 수익성 분석, 예산
14. ERP 인사 관리 (HRM) — 급여, 근태, 채용, 교육
15. ERP 생산 관리 (MRP, MES 연계) — 자재 소요 계획
16. ERP 구매 관리 — 전자 소싱, 공급자 관리
17. ERP 판매/유통 — 주문 관리, 수익 인식
18. ERP 마이그레이션 — 레거시→클라우드 ERP

---

## 3. SCM (공급망 관리) — 16개

1. SCM (Supply Chain Management) — 공급망 계획/실행/최적화
2. 공급망 구성 — 공급자/제조/유통/소매/소비자
3. SCM 핵심 프로세스 — Plan / Source / Make / Deliver / Return
4. SCOR 모델 (Supply Chain Operations Reference)
5. 채찍 효과 (Bullwhip Effect) — 수요 왜곡 증폭
6. 크로스 도킹 (Cross Docking) — 중간 보관 없는 유통
7. VMI (Vendor Managed Inventory) — 공급자 재고 관리
8. CPFR (Collaborative Planning, Forecasting and Replenishment)
9. JIT (Just-In-Time) — 무재고 생산, 도요타 생산 방식
10. 린 공급망 (Lean SCM) — 낭비 제거, 흐름 최적화
11. 수요 예측 (Demand Forecasting) — ARIMA, ML 기반
12. 공급망 가시성 (Supply Chain Visibility) — 실시간 추적
13. 디지털 공급망 (Digital Supply Chain) — IoT, AI, 블록체인
14. 공급망 리스크 관리 — 단일 공급처 위험, 다각화
15. 지속 가능 공급망 (Sustainable SCM) — ESG, 탄소 발자국
16. 스마트 물류 — 창고 자동화, AMR, 드론 배송

---

## 4. CRM (고객 관계 관리) — 14개

1. CRM (Customer Relationship Management) — 고객 생애 가치 극대화
2. CRM 유형 — 운영(Operational) / 분석(Analytical) / 협업(Collaborative) CRM
3. 고객 생애 가치 (CLV, Customer Lifetime Value)
4. 고객 여정 맵 (Customer Journey Map)
5. Salesforce CRM — 세일즈/서비스/마케팅 클라우드
6. 마케팅 자동화 (Marketing Automation) — Marketo, HubSpot
7. 옴니채널 (Omni-Channel) — 온오프라인 통합 고객 경험
8. CDP (Customer Data Platform) — 단일 고객 뷰
9. 챗봇 / AI 상담 (AI Customer Service)
10. NPS (Net Promoter Score) — 고객 추천 지수
11. 고객 세분화 (Customer Segmentation) — RFM 분석
12. 개인화 (Personalization) — 추천 엔진, 맞춤형 마케팅
13. 소셜 CRM — SNS 데이터 통합
14. 고객 이탈 예측 (Churn Prediction) — ML 모델

## 5. MES / 스마트 제조 — 16개

1. MES (Manufacturing Execution System) — 제조 실행 시스템
2. MES 기능 — 생산 일정/추적/품질/자원 관리
3. MRP (Material Requirements Planning) — 자재 소요 계획
4. MRP II — 제조 자원 계획
5. APS (Advanced Planning and Scheduling) — 정밀 일정 최적화
6. OPC-UA (Open Platform Communications - UA) — 산업 데이터 표준
7. SCADA (Supervisory Control and Data Acquisition)
8. DCS (Distributed Control System) — 분산 제어
9. PLC (Programmable Logic Controller) — 현장 제어
10. 스마트 팩토리 (Smart Factory) — CPS+IoT+AI+5G
11. 인더스트리 4.0 (Industry 4.0) — 4차 산업혁명 제조
12. AMR (Autonomous Mobile Robot) — 자율 이동 로봇
13. 코봇 (Collaborative Robot, Cobot) — 협업 로봇
14. 디지털 트윈 (Digital Twin) — 생산 시뮬레이션
15. 예지 정비 (Predictive Maintenance) — AI+IoT 고장 예측
16. PLM (Product Lifecycle Management) — 제품 생애주기 관리

1. BI (Business Intelligence) — 의사결정 지원
2. KPI 대시보드 — 실시간 경영 지표 시각화

6. 데이터 마이닝 — 연관/군집/분류/예측/시계열
7. 예측 분석 (Predictive Analytics) — ML 기반
8. 처방 분석 (Prescriptive Analytics) — 최적화, 시뮬레이션
9. 데이터 스토리텔링 (Data Storytelling)
10. 실시간 대시보드 — Grafana, Kibana, Redash
11. 자연어 질의 (NLQ) — BI에서 자연어 사용
12. 증강 분석 (Augmented Analytics) — AI 보조 분석
13. 데이터 리터러시 (Data Literacy) — 조직 역량
14. 데이터 카탈로그 통합 BI — Collibra, Alation

## 7. 엔터프라이즈 통합 / 미들웨어 — 18개

1. EAI (Enterprise Application Integration) — 기업 애플리케이션 통합
2. ESB (Enterprise Service Bus) — 느슨한 결합, 메시지 라우팅
3. SOA (Service-Oriented Architecture) — 서비스 단위 재사용
4. MSA (Microservices Architecture) — 독립 배포, 서비스 분리
5. API 관리 (API Management) — 게이트웨이, 정책, 포털
6. 이벤트 드리븐 아키텍처 (EDA)  — AsyncAPI, Kafka
7. 메시지 큐 (MQ) — IBM MQ, RabbitMQ, Active MQ, Kafka
8. ETL 도구 — Informatica, Talend, Apache NiFi
9. 데이터 통합 플랫폼 — MuleSoft, Dell Boomi, Apigee
10. CQRS 패턴 — 명령/조회 분리
11. 이벤트 소싱 (Event Sourcing) — 상태 이력 저장
12. Saga 패턴 — 분산 트랜잭션, 보상 처리
13. 아웃박스 패턴 (Outbox Pattern) — 이중 쓰기 문제 해결
14. 서킷 브레이커 (Circuit Breaker) — 장애 전파 방지, Hystrix, Resilience4j
15. 서비스 디스커버리 (Service Discovery) — Consul, Eureka, etcd
16. 분산 설정 관리 — Spring Cloud Config, Consul
17. 서비스 메시 (Service Mesh) — Istio, Linkerd (사이드카 패턴)
18. RPA (Robotic Process Automation) — 업무 자동화, UiPath, Automation Anywhere

5. SOX (Sarbanes-Oxley Act) — 내부통제, IT 감사

7. ITIL v4 — 서비스 가치 시스템, DITS 가이드
8. SLA (Service Level Agreement) — 서비스 수준 협약
9. SLI / SLO / SLA — 지표/목표/협약
10. IT 감사 (IT Audit) — 내부/외부 감사, ISACA CISA
11. 위험 관리 (Risk Management) — ISO 31000, NIST RMF
12. 컴플라이언스 (Compliance) — 규제 준수, GDPR/개인정보보호법

15. 전자정부법 / 정보통신망법

17. BCMS (Business Continuity Management System) — ISO 22301
18. DRP (Disaster Recovery Plan) / BCP (Business Continuity Plan)
