# IT 경영 / 전략 / 법무 키워드 목록

정보통신기술사·컴퓨터응용시스템기술사 대비 IT 경영·전략·법무 전 영역 기술사 수준 키워드
> ⚡ 기술사는 **기술 전문가 + 경영 판단자 + 법적 책임자**로서의 역할이 요구됨

---

## 1. IT 전략 경영 — 22개

1. 경영정보시스템 (MIS) — 의사결정 지원, 운영/관리/전략 계층
2. 전략 기획 프레임워크 — SWOT / PESTLE / Porter 5 Forces
3. Porter의 5 Forces — 경쟁자/신규진입자/대체제/공급자/구매자
4. PESTLE 분석 — 정치/경제/사회/기술/법률/환경 거시 요인
5. SWOT 분석 — 강점/약점/기회/위협
6. 밸류 체인 (Value Chain) — Porter, 본원 활동 + 지원 활동
7. 비즈니스 모델 캔버스 (BMC) — 9블록, 가치 제안
8. 디지털 비즈니스 모델 — 플랫폼 / 구독 / 프리미엄 / 데이터 모델
9. 전략적 IT 정렬 (Strategic Alignment) — 비즈니스-IT 연계
10. Gartner 하이프 사이클 (Hype Cycle) — 기술 성숙도 곡선
11. 기술 채택 수명 주기 (TALC) — Innovators→Chasm→Mainstream
12. 혁신 딜레마 (Innovator's Dilemma) — Christensen, 파괴적 혁신
13. 플랫폼 비즈니스 — 네트워크 효과, 양면 시장
14. API 이코노미 (API Economy) — 오픈 비즈니스 생태계
15. 데이터 중심 경영 (Data-Driven Management)
16. IT 투자 관리 — PPM (Portfolio/Program/Project Management)
17. IT ROI 측정 — 유형 편익 (비용 절감·효율) + 무형 편익
18. TOGAF ADM (Architecture Development Method) — 9단계 사이클
19. 기업 아키텍처 성숙도 — ACMM / NASCIO
20. 클라우드 경영 전략 — Cloud First / Cloud Smart 정책
21. 디지털 트윈 경영 — 시뮬레이션 기반 의사결정
22. 탄소 중립 IT 전략 — Green ICT, ESG 규제 대응

---

## 2. 프로젝트 관리 (PM) 심화 — 24개

1. PMBOK 7판 — 8개 성과 영역, 12개 원칙, 모델-방법-아티팩트
2. 이해관계자 관리 — 식별/분석/참여 계획/참여 관리/모니터링
3. 범위 관리 — WBS, 요구사항 추적 행렬 (RTM)
4. 일정 관리 — CPM / PERT / Gantt / 주경로 (Critical Path)
5. 비용 관리 — EVM (Earned Value Management), CPI / SPI
6. CV (Cost Variance) = EV - AC
7. SV (Schedule Variance) = EV - PV
8. CPI (Cost Performance Index) = EV/AC
9. EAC (Estimate at Completion) = BAC/CPI
10. 품질 관리 — QA (프로세스 감사) / QC (결과 검사), 6Sigma
11. 6시그마 — DMAIC, 불량률 3.4ppm, 품질 비용
12. 위험 관리 — 정성/정량 분석, 몬테카를로 시뮬레이션
13. 의사소통 관리 — 커뮤니케이션 채널 수 = n(n-1)/2
14. 자원 관리 — 자원 평준화 vs 자원 평활화
15. 조달 관리 — 고정가/원가보상/단가계약, 입찰 전략
16. 계약 유형 — FFP / FP-EPA / CPFF / CPIF / CPPC / T&M
17. 변경 관리 (Change Management) — CCB, 변경 요청, 영향 분석
18. 형상 관리 (SCM) — 형상 항목, 기준선, 감사
19. 프로젝트 게이트 리뷰 — 단계별 Go/No-Go 결정
20. 애자일 혼합 (Hybrid) — Agile + Waterfall, 규모 vs 유연성
21. SAFe (Scaled Agile Framework) — PI Planning, ART
22. 프로그램/포트폴리오 관리 — 전략 정렬, 이익 실현
23. 프로젝트 실패 원인 — CHAOS Report 분석
24. PMO (Project Management Office) — 지원형/통제형/지시형

## 3. IT 서비스 관리 (ITSM) — 18개

1. ITIL v4 — 서비스 가치 시스템 (SVS), DITS
2. ITIL 4 가이딩 원칙 — 7가지 원칙 (가치 집중 등)
3. 서비스 가치 체인 (SVC) — 참여/설계·TB/취득/제공/지원
4. 서비스 데스크 (Service Desk) — 단일 접점 (SPOC)
5. 인시던트 관리 — 탐지/분류/진단/에스컬레이션/해결/종료
6. 문제 관리 (Problem Management) — 근본 원인 분석 (RCA), 알려진 오류 DB
7. 변경 인에이블먼트 (Change Enablement) — 변경 유형 (표준/일반/긴급)
8. 릴리즈 관리 (Release Management)
9. 구성 관리 (Configuration Management) — CMDB (구성 항목 데이터베이스)
10. SLA (Service Level Agreement)
11. OLA (Operational Level Agreement) — 내부 팀 간 합의
12. UC (Underpinning Contract) — 외부 공급자와 계약
13. 가용성 관리 — MTBF / MTTR / 가용성 공식
14. 용량 관리 — 서비스/컴포넌트/비즈니스 용량 관리
15. 연속성 관리 (ITSCM) — DR, BCP, 위기 대응
16. 보안 관리 — ISO 27001 연계, 인시던트 대응
17. 공급자 관리 (Supplier Management) — 4분면 분류
18. 지속 개선 (Continual Improvement) — 카이젠, 개선 등록부

---

## 4. IT 거버넌스 / 컴플라이언스 심화 — 20개

1. COBIT 2019 — 40개 거버넌스/관리 목표, 핵심 설계 요소
2. COBIT 도메인 — EDM(거버넌스) / APO·BAI·DSS·MEA(관리)
3. ISO 38500 — IT 거버넌스 국제 표준, 이사회 원칙
4. 내부 통제 (Internal Control) — COSO 프레임워크, 5요소
5. COSO 2017 — ERM (Enterprise Risk Management)
6. SOX (사베인스-옥슬리법) — 404조 내부통제, IT 감사
7. IT 감사 (IT Audit) — ISACA CISA, 통제 목표 검증
8. 정보보호 관리 체계 — ISMS (ISO 27001) / ISMS-P (한국)
9. 전자금융감독규정 — 금융 IT 보안 기준, MDD
10. 개인정보 처리 원칙 — 적법/공정/투명, 목적 제한, 데이터 최소화
11. 개인정보 영향 평가 (PIA/DPIA) — 고위험 처리 사전 평가
12. AI 거버넌스 — EU AI Act, NIST AI RMF 2.0
13. ESG (Environmental/Social/Governance) — 공시 의무, 탄소 규제
14. 공급망 실사 (Due Diligence) — EU CSDD Directive
15. 클라우드 거버넌스 — 멀티클라우드 정책, 비용 통제
16. 데이터 거버넌스 — 데이터 소유자, 스튜어드, 카탈로그, 품질
17. 사이버 보험 (Cyber Insurance) — 보장 범위, 인수 심사
18. 사고 보고 의무 — 개인정보 침해 72시간 통보 (GDPR)
19. 법적 디스커버리 (Legal Discovery) — e-Discovery, EDRM
20. 규제 기술 (RegTech) — 자동화 컴플라이언스, AML

## 5. 기술 경제성 / 조달 — 14개

1. TCO 분석 — 하드웨어+소프트웨어+인력+운영+폐기 비용
2. OPEX vs CAPEX — 클라우드(OPEX) vs 온프레미스(CAPEX) 결정
3. 구매 vs 임차 vs 클라우드 비교 분석
4. FinOps — 클라우드 비용 최적화, 엔지니어링+재무+비즈니스 협업
5. 조달 프로세스 — RFI / RFQ / RFP / 제안서 평가 / 계약 / 협상
6. 소프트웨어 라이선싱 — 영구/구독/SaaS/오픈소스, 컴플라이언스
7. 벤더 관리 — 기술 잠금 (Vendor Lock-in) 위험
8. 클라우드 종료 전략 (Cloud Exit) — 이식성, 데이터 반환권
9. 오픈소스 활용 전략 — 라이선스 (MIT/Apache/GPL/AGPL) 구분
10. SW 저작권 — 프로그램 저작권, OSS 라이선스 준수
11. SLA 협상 포인트 — 가용성/RTO/RPO/패널티/면책 조항
12. IT 아웃소싱 전략 — 선택적 아웃소싱, MSP (Managed Service Provider)
13. Hidden Cost 분석 — 마이그레이션/교육/통합/유지보수 비용
14. 경제적 수명 주기 — 하드웨어 교체 주기, 감가상각

## 6. IT 법규·지식재산 — 16개

1. 전자정부법 — 전자정부 서비스, 행정 정보 공동 이용
2. 정보통신망 이용 촉진 및 정보보호 등에 관한 법률
3. 전기통신사업법 — 통신사 역무, 설비 공동 이용
4. 전파법 — 주파수 분배, 형식 승인, 전파 환경 보호
5. 소프트웨어 진흥법 — SW 산업 지원, 공정 거래
6. 클라우드컴퓨팅법 — 클라우드 이용, 보안 인증
7. 지능정보화 기본법 — AI·빅데이터 활성화
8. 산업디지털전환촉진법
9. 저작권법 — 소프트웨어 저작권, 데이터베이스권
10. 특허법 — SW 특허, 알고리즘 특허 제한
11. 영업 비밀 보호 — 부정경쟁방지법 및 영업비밀보호법
12. 전자서명법 — 공인/사설 전자서명, 법적 효력
13. 전자상거래 소비자보호법
14. 국가보안법 관련 ICT — 기밀 정보 처리 기준
15. 정보보호산업법 — MSSP 요건, 보안 제품 인증
16. 개인정보보호위원회 — 감독 기관, 제재 권한

## 7. 기술 동향 / 미래 전망 — 14개

1. 기술 예측 방법론 — 델파이 / 시나리오 / 특허 분석 / Gartner 사이클
2. 4차 산업혁명 핵심 기술 — AI / IoT / 빅데이터 / 클라우드 / 블록체인 / 자율주행 / 5G
3. Industry 5.0 — 인간 중심, 지속 가능, 회복 탄력성
4. 초연결 사회 (Hyper-Connected Society)
5. 디지털-물리 융합 (CPS/IoT+AI)
6. 양자 컴퓨팅 실용화 로드맵 — NISQ → 내결함 양자
7. 우주 인터넷 — LEO 위성 통신, Starlink v2
8. 6G 상용화 — 2030년대, 테라헤르츠, 지능형 반사면 (RIS)
9. 뇌-컴퓨터 인터페이스 (BCI) — Neuralink, OpenBCI
10. 나노기술+ICT 융합 — 나노 센서, 분자 통신
11. 디지털 트윈 고도화 — 국가 디지털 트윈, 스마트 시티
12. 소버린 AI / 소버린 클라우드 — 국가 주권 기술 전략
13. 오픈소스 생태계 전략 — 기여·커뮤니티·라이선스 전략
14. 인재 전략 — 디지털 인재 양성, 재교육 (Reskilling/Upskilling)
