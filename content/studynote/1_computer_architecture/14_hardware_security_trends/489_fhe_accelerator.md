# 465. DNA 저장 (DNA Storage)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: DNA 분자를 이용한 초고밀도 데이터 저장
> 2. **가치**: 수명, 밀도, 에너지 효율
> 3. **융합**: 합성, 인코딩, 암호화, 보존

---

## Ⅰ. 개요

### 개념 정의

DNA 저장(DNA Storage)은 **합성 DNA 분자를 디지털 데이터 저장 매체로 활용하여, 높은 밀도와 장기 보존을 가능하게 하는 기술**이다.

### 💡 비유: 생물학적 하드디스크

DNA 저장은 **생물학적 하드디스크**와 같다. 데이터가 있다. (디지털) 하드디스크가 있다. (DNA) 저장한다. (코딩) 꺼낸다. (읽기) 오래 보존된다. (안정성) 작다. (밀도) DNA 저장이 똑같다. 분자 기반, 초고밀도, 장기 보존.

## Ⅳ. 실무 적용

### DNA 저장 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                DNA 저장 분석                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【DNA 코딩 예시 (Python)】                                                    │)
│  ──────────────────                                                  │
│  # DNA 코딩 라이브러리 (이론적)                                              │
│  def encode_to_dna(binary_data):                                           │
│      mapping = {'00': 'A', '01': 'T', '10': 'G', '11': 'C'}                 │
│      dna = ''                                                                │
│      for i in range(0, len(binary_data), 2):                                │
│          base = binary_data[i:i+2]                                           │
│          dna += mapping[base]                                               │
│      return dna                                                              │
│                                                                     │
│  def decode_from_dna(dna):                                                  │
│      mapping = {'A': '00', 'T': '01', 'G': '10', 'C': '11'}                 │
│      return ''.join(mapping[base] for base in dna)                           │
│                                                                     │
│  【Microsoft Project Silica】                                                  │)
│  ──────────────────                                                  │
│  # DNA 저장 연구 프로젝트                                                   │
│  # 완전 자동화 DNA 저장 시스템 개발                                           │
│  # 목표: 수백 년 보관, 수십 PB 규모                                           │
│                                                                     │
│  【DNA 저장 팁】                                                                │)
│  ──────────────────                                                  │
│  1. 온도: -20°C 이하 냉동 보관                                               │
│  2. 오류 정정: Reed-Solomon 코드                                             │
│  3. 인덱싱: 빠른 검색을 위한 구조                                              │
│  4. 백업: 물리적 복제본 관리                                                  │
│  5. 보안: 접근 제어, 암호화                                                  │
│  6. 비용: 합성 비용 최적화                                                    │
└─────────────────────────────────────────────────────────────────────┘
```
