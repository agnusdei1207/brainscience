# 470. CXL (Compute Express Link)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: CPU-메모리-스토리지 간 고속 일관성 인터커넥트
> 2. **가치**: 메모리 풀링, 캐시 일관성, 유연한 확장
> 3. **융합**: PCIe, 캐시, 메모리, 스토리지

---

## Ⅰ. 개요

### 개념 정의

CXL(Compute Express Link)는 **PCIe 기반 고속 인터커넥트로, CPU와 가속기/메모리/스토리지 간 캐시 일관성과 메모리 일관성을 제공하는 표준**이다.

### 💡 비유: 고속 도로 네트워크
CXL은 **고속 도로 네트워크**와 같다. 도시가 있다. (CPU) 교외가 있다. (메모리) 산업단지가 있다. (스토리지) 도로가 연결한다. (CXL) 빠르다. 일관성 있다. 확장 가능. CXL이 똑같다. PCIe 기반, 캐시 일관성, 메모리 풀링.

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• CXL: Compute Express Link
• PCIe 기반: 캐시 일관성 인터커넥트
• CXL.io: I/O 프로토콜
• CXL.cache: 캐시 일관성
• CXL.mem: 메모리 접근
• 메모리 풀링: 공유 메모리 인프라
• Type 1/2/3: 장치 유형
• 장점: 메모리 확장, 일관성, 풀링
• 단점: 지연, 비용, 생태계
• 활용: 데이터센터, AI/ML, 서버
```
