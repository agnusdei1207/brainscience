# 464. 포토닉 컴퓨팅 (Photonic/Optical Computing)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 빛(광자)을 이용한 정보 처리 및 연산
> 2. **가치**: 대역폭, 에너지 효율, 지연 시간
> 3. **융합**: 광통신, 광인터커넥트, 광신경망

---

## Ⅰ. 개요

### 개념 정의

포토닉 컴퓨팅(Photonic Computing)은 **전자 대신 광자(빛)를 이용하여 정보를 처리, 전송, 저장하는 컴퓨팅 기술**이다.

### 💡 비유: 빛으로 통신하기

포토닉 컴퓨팅은 **빛으로 통신하기**와 같다. 전선이 있다. (전자) 느리다. 발열한다. 빛이 있다. (광자) 빠르다. 발열 없다. 신호등이 있다. (광 연산) 빛으로 계산한다. 포토닉이 똑같다. 빛 기반, 고속, 저전력.

### 포토닉 컴퓨팅 개요

```
┌─────────────────────────────────────────────────────────────────────┐
│                포토닉 컴퓨팅 개요                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【전자 vs 광자】                                                                │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  특성           전자(Electronic)      광자(Photonic)            │ │
│  │  ────           ───────────────      ─────────────             │ │
│  │  매체           전선                  광섬유/도파로              │ │
│  │  속도           ~1/3 c               c (광속)                  │ │
│  │  대역폭          제한적               THz급                      │ │
│  │  발열           높음                  낮음                       │ │
│  │  지연           저항/커패시턴스        매우 낮음                  │ │
│  │  간섭           전자기 간섭            광 간섭                    │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【광소자】                                                                      │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  소자              기능                    재료                   │ │
│  │  ────              ────                    ────                   │ │
│  │  레이저          광원                    InP, VCSEL              │ │
│  │  변조기          신호 변조                LiNbO3, InP            │ │
│  │  도파로          빛 경로                  Si, SiN                │ │
│  │  검출기          광→전 변환               Ge, InGaAs             │ │
│  │  스위치          경로 전환                MEMS, MZI              │ │
│  │  증폭기          광 신호 증폭              EDFA, SOA              │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【실리콘 포토닉스】                                                              │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  장점:                                                                  │ │
│  │  • CMOS 호환: 기존 반도체 공정 활용                                        │ │
│  │  • 집적도: 전자-광자 통합                                                  │ │
│  │  • 비용: 대량 생산 가능                                                   │ │
│  │                                                               │ │
│  │  도파로:                                                                │ │
│  │  • Si (n=3.47) / SiO2 (n=1.44) = 고굴절률 차                               │ │
│  │  • 1550nm: 통신 파장                                                      │ │
│  │  • 1310nm: 데이터센터 파장                                                │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

│  │                                                               │ │

│                                                                     │
│  【장단점】                                                                      │)
│  ──────────────────                                                  │

│  │                                                               │ │
│  │  장점:                                                                  │ │
