# 001. 운영체제 (Operating System)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 하드웨어와 사용자/애플리케이션 사이의 중개자
> 2. **가치**: 자원 관리, 편의성, 보호
> 3. **융합**: 커널, 시스템 콜, 드라이버, 프로세스

---

## Ⅰ. 개요

### 개념 정의

운영체제(Operating System, OS)는 **컴퓨터 하드웨어를 관리하고 사용자 및 응용 프로그램에 서비스를 제공하는 시스템 소프트웨어**다.

### 💡 비유: 정부
운영체제는 **정부**와 같다. 국가가 있다. (컴퓨터) 국민이 있다. (사용자/앱) 정부가 있다. (OS) 자원을 분배. (메모리, CPU) 법을 집행. (보호) 서비스 제공. (파일, 네트워크) OS가 똑같다. 자원 관리, 서비스 제공, 보호.

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 운영체제: 하드웨어와 사용자 사이 중개자
• 커널: OS 핵심, 특권 모드 실행
• 시스템 콜: 커널 서비스 요청 인터페이스
• 듀얼 모드: 사용자 모드 ↔ 커널 모드
• 자원 관리: CPU, 메모리, I/O, 파일
• 보호: 자원 격리, 권한
• 장점: 효율, 편의, 보안
• 단점: 오버헤드, 복잡성
• 예시: Linux, Windows, macOS
```
