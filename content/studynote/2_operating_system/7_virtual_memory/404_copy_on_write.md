# 404. Copy-on-Write (COW)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 쓰기 발생 시 페이지 복사
> 2. **가치**: 메모리 절약과 빠른 프로세스 생성
> 3. **융합**: 페이지 공유, fork와 연관

---

## Ⅰ. 개요

### 개념 정의

Copy-on-Write(COW)는 **여러 프로세스가 페이지를 공유하다가, 쓰기 발생 시 복사본을 만드는 기법**이다. 읽기는 공유하고 쓰기는 복사한다.

### 💡 비유: 복사기
COW는 **복사기**와 같다. 원본은 하나다. 누군가 수정하려면 복사본을 만든다. 원본은 그대로 둔다.

### COW 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                Copy-on-Write 구조                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【기본 원리】                                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  1. 초기 상태 (공유):                                                  │ │
│  │                                                             │ │
│  │  프로세스 A                     프로세스 B                            │ │
│  │  ┌─────────────┐              ┌─────────────┐                     │ │
│  │  │ PT: P0→F5  │              │ PT: P0→F5  │                     │ │
│  │  │     (R/O)  │              │     (R/O)  │  ← 읽기 전용!         │ │
│  │  └─────────────┘              └─────────────┘                     │ │
│  │         │                              │                           │ │
│  │         └──────────────┬───────────────┘                           │ │
│  │                        │                                           │ │
│  │                        ▼                                           │ │
│  │                    ┌───────┐                                        │ │
│  │                    │ F5    │  ← 하나의 프레임만 사용                   │ │
│  │                    └───────┘                                        │ │
│  │                                                             │ │
│  │  2. 쓰기 발생 시:                                                       │ │
│  │     • 페이지 폴트 발생 (쓰기 보호 위반)                                 │ │
│  │     • 새 프레임 할당                                                   │ │
│  │     • 내용 복사                                                       │ │
│  │     • 페이지 테이블 갱신                                                │ │
│  │     • 쓰기 권한 부여                                                   │ │
│  │                                                             │ │
│  │  3. COW 후 상태:                                                       │ │
│  │                                                             │ │
│  │  프로세스 A                     프로세스 B                            │ │
│  │  ┌─────────────┐              ┌─────────────┐                     │ │
│  │  │ PT: P0→F5  │              │ PT: P0→F10 │                     │ │
│  │  │     (R/O)  │              │     (R/W)  │  ← 쓰기 가능         │ │
│  │  └─────────────┘              └─────────────┘                     │ │
│  │         │                              │                           │ │
│  │         │                              │                           │ │
│  │         ▼                              ▼                           │ │
│  │     ┌───────┐                     ┌───────┐                        │ │
│  │     │ F5    │                     │ F10   │  ← 복사본               │ │
│  │     └───────┘                     └───────┘                        │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                Copy-on-Write 상세                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【fork()에서의 COW】                                                     │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  fork() 시:                                                           │ │
│  │  • 모든 페이지를 복사하지 않음                                          │ │
│  │  • 페이지 테이블만 복사                                                │ │
│  │  • 모든 페이지를 읽기 전용으로 설정                                     │ │
│  │  • 참조 카운트 증가                                                    │ │
│  │                                                             │ │
│  │  장점:                                                                 │ │
│  │  • fork() 매우 빠름                                                   │ │
│  │  • 메모리 절약                                                         │ │
│  │  • 명령어 바로 호출 시 대부분 복사 불필요                              │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【참조 카운트】                                                            │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  각 프레임에 참조 카운트 유지:                                          │ │
│  │                                                             │ │
│  │  • fork(): 카운트 증가                                                │ │
│  │  • COW: 원본 카운트 감소, 새 페이지 카운트 = 1                         │ │
│  │  • exit(): 카운트 감소, 0이면 해제                                    │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Linux COW 동작】                                                        │
│  ──────────────────                                                  │
│  // fork() 후 메모리 사용 비교                                           │
│  // fork 직후: shared가 같음 (COW로 공유 중)                            │
│                                                                     │
│  // 자식이 쓰기 후 shared 감소 = COW로 복사됨                              │
│                                                                     │
│  【COW 페이지 폴트 처리】                                                   │
│  ──────────────────                                                  │
│  // 커널 코드 (개념적)                                                    │
│  void handle_cow_fault(PageTableEntry pte) {                        │
│      Frame frame = pte.frame;                                        │
│                                                                     │
│      if (frame.ref_count == 1) {                                    │
│          // 유일한 소유자: 그냥 쓰기 권한 부여                             │ │
│          pte.writable = true;                                        │ │
│      } else {                                                       │
│          // 다른 프로세스도 공유 중: 복사 필요                              │ │
│          Frame new_frame = allocate_frame();                         │ │
│          copy_page(frame, new_frame);                                │ │
│          frame.ref_count--;                                         │
│          pte.frame = new_frame;                                     │
│          new_frame.ref_count = 1;                                   │
│          pte.writable = true;                                        │ │
│      }                                                              │
│  }                                                                  │
│                                                                     │
│  【파일 시스템 COW】                                                        │
│  ──────────────────                                                  │
│  // Btrfs, ZFS 등 COW 파일 시스템                                       │
│  // 파일 수정 시 새 블록 할당                                              │
│  // 이전 버전 유지 (스냅샷 가능)                                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
