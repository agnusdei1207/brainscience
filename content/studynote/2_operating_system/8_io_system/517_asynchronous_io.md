# 517. 비동기 I/O (Asynchronous I/O)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: I/O 완료 대기 없이 즉시 반환
> 2. **가치**: 프로그램 응답성과 병렬 처리
> 3. **융합**: 논블로킹, 이벤트 루프, 콜백과 연관

---

## Ⅰ. 개요

### 개념 정의

비동기 I/O(Asynchronous I/O, AIO)는 **I/O 작업을 시작한 후 완료를 기다리지 않고 즉시 반환하여 다른 작업을 계속할 수 있는 방식**이다.

### 💡 비유: 음식점 주문
비동기 I/O는 **음식점에서 주문 후 진동벨을 받는 것**과 같다. 음식이 나올 때까지 기다리지 않고 다른 일을 한다.

### 비동기 I/O 구조

├─────────────────────────────────────────────────────────────────────┤

│  【동기 vs 비동기】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  동기 I/O (Synchronous):                                                │ │
│  │  ┌─────────────────────────────────────────────────────────┐    │ │
│  │  │  스레드: read() ──────────── 대기 ──────────── 완료       │    │ │
│  │  │                      (블로킹)                            │    │ │
│  │  │                                                          │    │ │
│  │  │  CPU 활용: 낮음 (대기 시간)                                │    │ │
│  │  └─────────────────────────────────────────────────────────┘    │ │
│  │                                                             │ │
│  │  비동기 I/O (Asynchronous):                                              │ │
│  │  ┌─────────────────────────────────────────────────────────┐    │ │
│  │  │  스레드: aio_read() → 즉시 반환 → 다른 작업 → 완료 알림      │    │ │
│  │  │                      (논블로킹)                            │    │ │
│  │  │                                                          │    │ │
│  │  │  CPU 활용: 높음 (대기 없이 다른 작업 수행)                   │    │ │
│  │  └─────────────────────────────────────────────────────────┘    │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【비동기 I/O 모델】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  모델              알림 방식                예시                           │ │
│  │  ────              ────────                ────                           │ │
│  │  콜백              함수 호출                Node.js                       │ │
│  │  이벤트            이벤트 큐                libuv, Windows                │ │
│  │  시그널            SIGIO                   POSIX AIO                     │ │
│  │  폴링              aio_error()             POSIX AIO                     │ │
│  │  완료 포트         IOCP                    Windows                       │ │
│  │  io_uring          완료 큐                 Linux 5.1+                    │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【POSIX AIO 예시】                                                      │
│  ──────────────────                                                  │
│  #include <aio.h>                                                    │
│  #include <fcntl.h>                                                  │
│  #include <unistd.h>                                                 │
│                                                                     │
│  int main() {                                                        │
│      int fd = open("file.txt", O_RDONLY);                            │
│      char buf[4096];                                                 │
│      struct aiocb aio = {0};                                         │
│                                                                     │
│      // AIO 구조체 설정                                                 │
│      aio.aio_fildes = fd;                                            │
│      aio.aio_buf = buf;                                              │
│      aio.aio_nbytes = sizeof(buf);                                   │
│      aio.aio_offset = 0;                                             │
│                                                                     │
│      // 비동기 읽기 시작                                                 │
│      aio_read(&aio);                                                 │
│                                                                     │
│      // 다른 작업 수행...                                                │
│      printf("Reading in background...\n");                           │
│                                                                     │
│      // 완료 대기                                                        │
│      while (aio_error(&aio) == EINPROGRESS) {                       │
│          // 대기 또는 다른 작업                                           │
│      }                                                               │
│                                                                     │
│      // 결과 확인                                                       │
│      ssize_t bytes = aio_return(&aio);                               │
│      printf("Read %zd bytes\n", bytes);                              │
│                                                                     │
│      close(fd);                                                      │
│      return 0;                                                        │
│  }                                                                   │
│                                                                     │
│  【libaio (Linux 네이티브 AIO)】                                         │
│  ──────────────────                                                  │
│  #include <libaio.h>                                                 │
│                                                                     │
│  io_context_t ctx = 0;                                               │
│  io_queue_init(128, &ctx);                                           │
│                                                                     │
│  struct iocb cb;                                                     │
│  struct iocb *cbs[1] = {&cb};                                        │
│                                                                     │
│  // 읽기 준비                                                          │
│  io_prep_pread(&cb, fd, buf, size, offset);                          │
│  cb.data = (void*)"my_data";  // 사용자 데이터                          │
│                                                                     │
│  // 제출                                                              │
│  io_submit(ctx, 1, cbs);                                             │
│                                                                     │
│  // 완료 대기                                                          │
│  struct io_event events[1];                                          │
│  io_getevents(ctx, 1, 1, events, NULL);                              │
│                                                                     │
│  // 결과 처리                                                          │
│  printf("Read %ld bytes\n", events[0].res);                          │
│                                                                     │
│  io_queue_release(ctx);                                              │
│                                                                     │
│  【io_uring 예시】                                                        │
│  ──────────────────                                                  │
│  #include <liburing.h>                                               │
│                                                                     │
│  struct io_uring ring;                                               │
│  io_uring_queue_init(32, &ring, 0);                                  │
│                                                                     │
│  // 제출 엔트리 획득                                                     │
│  struct io_uring_sqe *sqe = io_uring_get_sqe(&ring);                 │
│  io_uring_prep_read(sqe, fd, buf, size, offset);                     │
│  sqe->user_data = (unsigned long)buf;                                │
│                                                                     │
│  // 제출                                                              │
│  io_uring_submit(&ring);                                             │
│                                                                     │
│  // 완료 대기                                                          │
│  struct io_uring_cqe *cqe;                                           │
│  io_uring_wait_cqe(&ring, &cqe);                                     │
│                                                                     │
│  // 결과 확인                                                          │
│  int bytes_read = cqe->res;                                          │
│  io_uring_cqe_seen(&ring, cqe);                                      │
│                                                                     │
│  io_uring_queue_exit(&ring);                                         │
│                                                                     │
│  【Node.js 비동기 I/O】                                                  │
│  ──────────────────                                                  │
│  const fs = require('fs');                                           │
│                                                                     │
│  // 콜백 방식                                                          │
│  fs.readFile('file.txt', (err, data) => {                            │
│      if (err) throw err;                                             │
│      console.log('Read:', data.length);                              │
│  });                                                                 │
│  console.log('Reading...');  // 먼저 실행                              │
│                                                                     │
│  // Promise 방식                                                      │
│  const data = await fs.promises.readFile('file.txt');                │
│  console.log('Read:', data.length);                                  │
│                                                                     │
│  【Python 비동기 I/O (asyncio)】                                         │
│  ──────────────────                                                  │
│  import aiofiles                                                     │
│  import asyncio                                                      │
│                                                                     │
│  async def read_file():                                              │
│      async with aiofiles.open('file.txt', 'r') as f:                 │
│          content = await f.read()                                    │
│          print(f'Read: {len(content)}')                              │
│                                                                     │
│  asyncio.run(read_file())                                            │
│                                                                     │
│  【io_uring 지원 확인】                                                  │
│  ──────────────────                                                  │
│  // 커널 버전 확인                                                       │
│  $ uname -r                                                          │
│  5.15.0                                                              │
│                                                                     │
│  // io_uring 지원 확인                                                  │
│  $ cat /proc/sys/kernel/io_uring_disabled                           │
│  0  // 0 = 활성화                                                      │
│                                                                     │
│  // liburing 설치                                                      │
│  $ sudo apt install liburing-dev                                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: I/O 완료 대기 없이 즉시 반환
• 비교: 동기(블로킹) vs 비동기(논블로킹)
• API: POSIX AIO, libaio, io_uring
• 알림: 콜백, 이벤트, 시그널, 폴링
• io_uring: 고성능 Linux 비동기 I/O
• 장점: 응답성, 리소스 효율, 높은 동시성
• 단점: 복잡도, 디버깅 어려움
• 언어: Node.js, Python asyncio, Rust async
```
