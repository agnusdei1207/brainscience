# 443. 디렉토리 (Directory)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파일을 계층적으로 조직화
> 2. **가치**: 파일 탐색과 관리 용이
> 3. **융합**: 파일 시스템, 경로와 연관

---

## Ⅰ. 개요

### 개념 정의

디렉토리(Directory)는 **파일과 하위 디렉토리를 계층적으로 조직화하는 컨테이너**다. 폴더라고도 한다.

### 💡 비유: 서랍장
디렉토리는 **서랍장**과 같다. 서랍에 문서를 넣는다. 서랍 안에 작은 서랍도 넣을 수 있다.

### 디렉토리 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                디렉토리 구조                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【계층적 구조】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                         / (root)                                 │ │
│  │                          │                                         │ │
│  │          ┌───────────────┼───────────────┐                        │ │
│  │          │               │               │                        │ │
│  │          ▼               ▼               ▼                        │ │
│  │       ┌─────┐        ┌─────┐        ┌─────┐                     │ │
│  │       │ home│        │ etc │        │ var │                     │ │
│  │       └──┬──┘        └──┬──┘        └──┬──┘                     │ │
│  │          │               │               │                        │ │
│  │          ▼               ▼               ▼                        │ │
│  │       ┌─────┐        ┌─────┐        ┌─────┐                     │ │
│  │       │ user│        │passwd│       │ log │                     │ │
│  │       └──┬──┘        └─────┘        └─────┘                     │ │
│  │          │                                                        │ │
│  │          ▼                                                        │ │
│  │       ┌─────┐                                                   │ │
│  │       │ doc │                                                   │ │
│  │       └─────┘                                                   │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【디렉토리 항목】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  디렉토리 = (파일이름, 아이노드번호) 목록                                 │ │
│  │                                                             │ │
│  │  예: /home/user/ 디렉토리                                               │ │
│  │  ┌─────────────────────────────────────────────────────────┐  │ │
│  │  │ 파일 이름          아이노드 번호    파일 형식                  │  │ │
│  │  │ .                  12345         디렉토리 (자신)             │  │ │
│  │  │ ..                 12340         디렉토리 (부모)             │  │ │
│  │  │ document.txt       12400         일반 파일                   │  │ │
│  │  │ image.png          12401         일반 파일                   │  │ │
│  │  │ subfolder/         12402         디렉토리                    │  │ │
│  │  └─────────────────────────────────────────────────────────┘  │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                디렉토리 상세                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【절대 경로 vs 상대 경로】                                            │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  절대 경로 (Absolute Path):                                             │ │
│  │  • 루트(/)부터 시작                                                      │ │
│  │  • 예: /home/user/document.txt                                         │ │
│  │                                                             │ │
│  │  상대 경로 (Relative Path):                                              │ │
│  │  • 현재 디렉토리부터 시작                                                 │ │
│  │  • 예: ./document.txt, ../parent/file.txt                              │ │
│  │                                                             │ │
│  │  특수 디렉토리:                                                           │ │
│  │  • . : 현재 디렉토리                                                      │ │
│  │  • .. : 부모 디렉토리                                                     │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【디렉토리 구현】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  유닉스/Linux:                                                           │ │
│  │  • 디렉토리도 파일의 일종                                                 │ │
│  │  • (파일이름, 아이노드) 쌍의 리스트                                       │ │
│  │  • 특수 시스템콜: opendir, readdir, closedir                           │ │
│  │                                                             │ │
│  │  Windows (NTFS):                                                         │ │
│  │  • B+ Tree 구조                                                          │ │
│  │  • 파일 이름 유니코드                                                    │ │
│  │  • 대소문자 구분 안 함                                                    │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【링크】                                                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  하드 링크 (Hard Link):                                                  │ │
│  │  • 같은 아이노드 가리킴                                                   │ │
│  │  • 원본 삭제해도 링크 유효                                                │ │
│  │  • 같은 파일 시스템 내에서만                                              │ │
│  │                                                             │ │
│  │  심볼릭 링크 (Symbolic/Soft Link):                                       │ │
│  │  • 파일 경로 저장                                                        │ │
│  │  • 원본 삭제 시 링크 끊김                                                 │ │
│  │  • 다른 파일 시스템 가능                                                  │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【디렉토리 조작】                                                      │
│  ──────────────────                                                  │
│  // 디렉토리 생성                                                        │
│  $ mkdir newdir                                                       │
│  $ mkdir -p path/to/nested/dir                                        │
│                                                                     │
│  // 디렉토리 삭제                                                        │
│  $ rmdir emptydir                                                    │
│  $ rm -r directory                                                   │
│                                                                     │
│  // 디렉토리 이동/이름변경                                                │
│  $ mv olddir newdir                                                  │
│                                                                     │
│  // 디렉토리 내용 확인                                                    │
│  $ ls                                                                │
│  $ ls -la                                                            │
│  $ tree                                                              │
│                                                                     │
│  // 현재 디렉토리 확인                                                     │
│  $ pwd                                                               │
│  /home/user                                                          │
│                                                                     │
│  // 디렉토리 변경                                                         │
│  $ cd /home/user                                                     │
│  $ cd ..                                                             │
│  $ cd -                                                              │
│                                                                     │
│  【링크 생성】                                                            │
│  ──────────────────                                                  │
│  // 하드 링크                                                            │
│  $ ln original.txt hardlink.txt                                      │
│                                                                     │
│  // 심볼릭 링크                                                           │
│  $ ln -s original.txt symlink.txt                                    │
│                                                                     │
│  // 링크 확인                                                            │
│  $ ls -la                                                            │
│  lrwxrwxrwx 1 user user 11 Mar 8 10:00 symlink.txt -> original.txt  │
│  -rw-r--r-- 2 user user 100 Mar 8 10:00 original.txt                 │
│  -rw-r--r-- 2 user user 100 Mar 8 10:00 hardlink.txt                 │
│                                                                     │
│  【프로그래밍에서 디렉토리 다루기】                                        │
│  ──────────────────                                                  │
│  #include <dirent.h>                                                 │
│  #include <sys/stat.h>                                               │
│                                                                     │
│  // 디렉토리 읽기                                                         │
│  DIR *dir = opendir("/home/user");                                   │
│  struct dirent *entry;                                               │
│  while ((entry = readdir(dir)) != NULL) {                            │
│      printf("%s (inode: %ld)\n", entry->d_name, entry->d_ino);        │
│  }                                                                   │
│  closedir(dir);                                                       │
│                                                                     │
│  // 디렉토리 생성                                                        │
│  mkdir("/tmp/newdir", 0755);                                          │
│                                                                     │
│  // 현재 작업 디렉토리                                                     │
│  char cwd[PATH_MAX];                                                 │
│  getcwd(cwd, sizeof(cwd));                                           │
│  chdir("/home/user");                                                │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 파일을 계층적으로 조직화하는 컨테이너

```
