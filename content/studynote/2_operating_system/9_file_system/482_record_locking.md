# 482. 레코드 잠금 (Record Locking)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파일 내 특정 범위(레코드) 잠금
> 2. **가치**: 세밀한 동시성 제어
> 3. **융합**: 파일 잠금, 데이터베이스와 연관

---

## Ⅰ. 개요

### 개념 정의

레코드 잠금(Record Locking)은 **파일 전체가 아닌 특정 바이트 범위(레코드)만 잠그는 방식**이다.

### 💡 비유: 도서관 책의 특정 페이지
레코드 잠금은 **도서관 책의 특정 페이지**와 같다. 한 사람이 10페이지를 보는 동안 다른 사람은 20페이지를 볼 수 있다.

### 레코드 잠금 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                레코드 잠금                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【파일 잠금 vs 레코드 잠금】                                          │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  파일 잠금 (flock):                                                   │ │
│  │  ┌─────────────────────────────────────────────────────────┐    │ │
│  │  │  ┌───────────────────────────────────────────────────┐ │    │ │
│  │  │  │                 전체 파일 잠금                     │ │    │ │
│  │  │  │  [============================================]  │ │    │ │
│  │  │  └───────────────────────────────────────────────────┘ │    │ │
│  │  │  다른 프로세스는 파일 전체에 접근 불가                      │    │ │
│  │  └─────────────────────────────────────────────────────────┘    │ │
│  │                                                             │ │
│  │  레코드 잠금 (fcntl):                                                  │ │
│  │  ┌─────────────────────────────────────────────────────────┐    │ │
│  │  │  ┌───────┐         ┌───────┐         ┌───────┐        │    │ │
│  │  │  │레코드 1│         │레코드 2│         │레코드 3│        │    │ │
│  │  │  │ 잠금  │         │       │         │ 잠금  │        │    │ │
│  │  │  └───────┘         └───────┘         └───────┘        │    │ │
│  │  │  다른 프로세스는 레코드 2에 접근 가능                        │    │ │
│  │  └─────────────────────────────────────────────────────────┘    │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【fcntl flock 구조체】                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  struct flock {                                                      │
│  │      short l_type;    // 잠금 타입                                     │ │
│  │                       // F_RDLCK: 읽기 잠금                            │ │
│  │                       // F_WRLCK: 쓰기 잠금                            │ │
│  │                       // F_UNLCK: 잠금 해제                            │ │
│  │                                                             │ │
│  │      short l_whence;  // 기준점                                        │ │
│  │                       // SEEK_SET: 파일 시작                            │ │
│  │                       // SEEK_CUR: 현재 위치                            │ │
│  │                       // SEEK_END: 파일 끝                              │ │
│  │                                                             │ │
│  │      off_t l_start;   // 시작 오프셋                                    │ │
│  │      off_t l_len;     // 잠금 길이                                      │ │
│  │                       // 0: l_start부터 파일 끝까지                     │ │
│  │      pid_t l_pid;     // 잠금 소유자 PID (F_GETLK)                      │ │
│  │  };                                                                   │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                레코드 잠금 상세                                        │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【잠금 범위 예시】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  파일 구조 (레코드당 100바이트):                                        │ │
│  │  ┌─────────┬─────────┬─────────┬─────────┬─────────┐          │ │
│  │  │레코드 0  │레코드 1  │레코드 2  │레코드 3  │레코드 4  │          │ │
│  │  │ 0-99    │100-199  │200-299  │300-399  │400-499  │          │ │
│  │  └─────────┴─────────┴─────────┴─────────┴─────────┘          │ │
│  │                                                             │ │
│  │  레코드 2 잠금:                                                        │ │
│  │  struct flock fl;                                                    │ │
│  │  fl.l_type = F_WRLCK;                                                │ │
│  │  fl.l_whence = SEEK_SET;                                             │ │
│  │  fl.l_start = 200;    // 레코드 2 시작                                │ │
│  │  fl.l_len = 100;      // 레코드 2 길이                                │ │
│  │  fcntl(fd, F_SETLK, &fl);                                            │ │
│  │                                                             │ │
│  │  전체 파일 잠금:                                                       │ │
│  │  fl.l_start = 0;                                                      │ │
│  │  fl.l_len = 0;         // 0 = 파일 끝까지                              │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【잠금 상속과 해제】                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  1. close() 시:                                                       │ │
│  │     • 해당 fd의 모든 잠금 해제                                          │ │
│  │     • 같은 파일의 다른 fd도 영향                                        │ │
│  │                                                             │ │
│  │  2. fork() 시:                                                        │ │
│  │     • 잠금은 상속되지 않음                                              │ │
│  │     • 자식은 새로 잠금 획득 필요                                        │ │
│  │                                                             │ │
│  │  3. dup() 시:                                                         │ │
│  │     • 잠금은 복사됨                                                    │ │
│  │     • 원본과 복사본이 같은 잠금 공유                                     │ │
│  │                                                             │ │
│  │  주의: 한 프로세스의 잠금은 그 프로세스의 다른 fd에도 영향               │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【잠금 충돌 해결】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  F_SETLK: 비차단, 실패 시 즉시 반환 (EACCES/EAGAIN)                   │ │
│  │  F_SETLKW: 차단, 잠금 획득까지 대기                                    │ │
│  │  F_GETLK: 잠금 정보 조회 (차단된 경우 누가 잡고 있는지)                 │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【레코드 잠금 함수 구현】                                            │
│  ──────────────────                                                  │
│  #include <fcntl.h>                                                  │
│  #include <unistd.h>                                                 │
│                                                                     │
│  // 레코드 잠금 획득                                                   │
│  int lock_record(int fd, int record_num, int record_size,           │
│                   int lock_type) {                                   │
│      struct flock fl;                                                │
│      fl.l_type = lock_type;  // F_RDLCK or F_WRLCK                  │
│      fl.l_whence = SEEK_SET;                                         │
│      fl.l_start = record_num * record_size;                          │
│      fl.l_len = record_size;                                         │
│      fl.l_pid = 0;                                                    │
│                                                                     │
│      return fcntl(fd, F_SETLKW, &fl);  // 차단 모드                   │
│  }                                                                   │
│                                                                     │
│  // 레코드 잠금 해제                                                   │
│  int unlock_record(int fd, int record_num, int record_size) {        │
│      struct flock fl;                                                │
│      fl.l_type = F_UNLCK;                                            │
│      fl.l_whence = SEEK_SET;                                         │
│      fl.l_start = record_num * record_size;                          │
│      fl.l_len = record_size;                                         │
│                                                                     │
│      return fcntl(fd, F_SETLK, &fl);                                 │
│  }                                                                   │
│                                                                     │
│  【사용 예시】                                                          │
│  ──────────────────                                                  │
│  #define RECORD_SIZE 100                                             │
│                                                                     │
│  int fd = open("database.dat", O_RDWR);                              │
│                                                                     │
│  // 레코드 5에 쓰기 잠금                                               │
│  if (lock_record(fd, 5, RECORD_SIZE, F_WRLCK) == -1) {              │
│      perror("lock failed");                                           │
│      exit(1);                                                         │
│  }                                                                   │
│                                                                     │
│  // 레코드 수정                                                        │
│  lseek(fd, 5 * RECORD_SIZE, SEEK_SET);                               │
│  write(fd, data, RECORD_SIZE);                                       │
│                                                                     │
│  // 잠금 해제                                                          │
│  unlock_record(fd, 5, RECORD_SIZE);                                  │
│                                                                     │
│  close(fd);  // 모든 잠금 자동 해제                                    │
│                                                                     │
│  【잠금 상태 확인】                                                    │
│  ──────────────────                                                  │
│  int check_lock(int fd, off_t start, off_t len) {                    │
│      struct flock fl;                                                │
│      fl.l_type = F_WRLCK;                                            │
│      fl.l_whence = SEEK_SET;                                         │
│      fl.l_start = start;                                              │
│      fl.l_len = len;                                                  │
│                                                                     │
│      if (fcntl(fd, F_GETLK, &fl) == -1) {                            │
│          return -1;  // 에러                                          │ │
│      }                                                                │
│                                                                     │
│      if (fl.l_type == F_UNLCK) {                                     │
│          return 0;   // 잠금 없음                                      │ │
│      } else {                                                          │
│          printf("Locked by PID %d\n", fl.l_pid);                     │
│          return 1;   // 잠금 있음                                      │ │
│      }                                                                │
│  }                                                                   │
│                                                                     │
│  【비차단 잠금 시도】                                                  │
│  ──────────────────                                                  │
│  int try_lock(int fd, off_t start, off_t len) {                      │
│      struct flock fl;                                                │
│      fl.l_type = F_WRLCK;                                            │
│      fl.l_whence = SEEK_SET;                                         │
│      fl.l_start = start;                                              │
│      fl.l_len = len;                                                  │
│                                                                     │
│      if (fcntl(fd, F_SETLK, &fl) == -1) {                            │
│          if (errno == EACCES || errno == EAGAIN) {                    │
│              return 0;  // 이미 잠김                                   │ │
│          }                                                             │
│          return -1;  // 에러                                          │ │
│      }                                                                │
│      return 1;  // 잠금 획득 성공                                      │ │
│  }                                                                   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 파일 내 특정 바이트 범위 잠금
• 구조체: struct flock (l_type, l_start, l_len)
• 타입: F_RDLCK, F_WRLCK, F_UNLCK
• 명령: F_SETLK (비차단), F_SETLKW (차단), F_GETLK
• 장점: 세밀한 동시성, 높은 병렬성
• 해제: close() 시 자동 해제
• fork: 잠금 상속 안 됨
• 용도: 데이터베이스, 공유 데이터 파일
```
