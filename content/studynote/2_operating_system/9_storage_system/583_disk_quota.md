# 583. 디스크 할당량 (Disk Quota)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 사용자/그룹별 디스크 사용량 제한
> 2. **가치**: 공정한 자원 분배, 시스템 안정성
> 3. **융합**: 파일시스템, 사용자 관리, 모니터링과 연관

---

## Ⅰ. 개요

### 개념 정의
**디스크 할당량(Disk Quota)**은 **사용자나 그룹의 디스크 사용량을 제한하는 기능**입니다.

### 💡 비유: 식당 식권
디스크 할당량은 **식당 식권**과 같습니다. 한 사람이 먹을 수 있는 양을 제한합니다.

### 디스크 할당량 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                디스크 할당량 구조                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【할당량 종류】                                                       │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  종류              제한 대상              설명                   │ │   │
│  │  ────              ────────              ────                   │ │   │
│  │  사용자 할당량       개별 사용자             UID 기반               │ │   │
│  │  그룹 할당량         그룹                  GID 기반               │ │   │
│  │  프로젝트 할당량      프로젝트 디렉토리         디렉토리 기반            │ │   │
│  │                                                             │ │   │
│  │  제한 단위:                                                      │ │   │
│  │  • 블록 할당량: 디스크 공간 (KB)                                  │ │   │
│  │  • inode 할당량: 파일 수                                         │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【소프트/하드 제한】                                                  │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  제한 유형         설명                    동작                   │ │   │
│  │  ────────         ────                    ────                   │ │   │
│  │  Soft Limit       경고 임계값              초과 시 경고             │ │   │
│  │                   (유예 기간 있음)           유예 기간 내 허용         │ │   │
│  │                                                             │ │   │
│  │  Hard Limit       절대 최대값              초과 불가               │ │   │
│  │                   (유예 기간 없음)           즉시 차단               │ │   │
│  │                                                             │ │   │
│  │  Grace Period     유예 기간                 soft limit 초과 후       │ │   │
│  │                   (기본 7일)               hard limit 적용 전        │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【할당량 파일시스템 지원】                                             │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  파일시스템       할당량 지원             명령어                    │ │   │
│  │  ────────         ─────────             ──────                    │ │   │
│  │  ext4            quota, usrquota        quota, edquota            │ │   │
│  │  XFS             자체 지원               xfs_quota                 │ │   │
│  │  Btrfs           qgroup                  btrfs qgroup              │ │   │
│  │  ZFS             자체 지원               zfs set quota             │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                디스크 할당량 상세                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【할당량 파일】                                                       │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  ext4 할당량 파일:                                               │ │   │
│  │  • aquota.user   - 사용자 할당량                                 │ │   │
│  │  • aquota.group  - 그룹 할당량                                   │ │   │
│  │                                                             │ │   │
│  │  XFS: 파일시스템 내부에 저장                                      │ │   │
│  │                                                             │ │   │
│  │  할당량 파일 생성:                                                │ │   │
│  │  $ sudo quotacheck -cug /mnt/data                              │ │   │
│  │  // -c: 생성, -u: 사용자, -g: 그룹                                │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【할당량 설정값】                                                     │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  edquota 출력 예시:                                              │ │   │
│  │  ┌────────────────────────────────────────────────────┐        │ │   │
│  │  │ Filesystem  blocks  soft  hard  inodes  soft  hard │        │ │   │
│  │  │ /dev/sda1   50000   90000  100000  1200   0     0  │        │ │   │
│  │  └────────────────────────────────────────────────────┘        │ │   │
│  │                                                             │ │   │
│  │  blocks: 현재 사용 블록 (1KB)                                    │ │   │
│  │  soft/hard (blocks): 블록 제한 (KB)                              │ │   │
│  │  inodes: 현재 파일 수                                            │ │   │
│  │  soft/hard (inodes): 파일 수 제한                                 │ │   │
│  │                                                             │ │   │
│  │  0 = 제한 없음                                                    │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【유예 기간 (Grace Period)】                                          │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  기본: 7일                                                       │ │   │
│  │                                                             │ │   │
│  │  // 유예 기간 확인/설정                                           │ │   │
│  │  $ sudo edquota -t                                             │ │   │
│  │  Filesystem     Block grace period  Inode grace period         │ │   │
│  │  /dev/sda1      7days               7days                      │ │   │
│  │                                                             │ │   │
│  │  // 단위: seconds, minutes, hours, days, weeks                  │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【ext4 할당량 설정】                                                 │
│  ──────────────────                                                │
│  // 1. 마운트 옵션 추가                                               │
│  $ sudo mount -o usrquota,grpquota /dev/sda1 /home                 │
│  // 또는 /etc/fstab                                                  │
│  /dev/sda1  /home  ext4  defaults,usrquota,grpquota  0  2          │
│  $ sudo mount -o remount /home                                     │
│                                                                     │
│  // 2. 할당량 파일 생성                                                │
│  $ sudo quotacheck -cug /home                                      │
│  $ sudo quotacheck -avug             // 전체 검사                    │
│                                                                     │
│  // 3. 할당량 활성화                                                  │
│  $ sudo quotaon /home                                               │
│  $ sudo quotaon -a                  // 전체 활성화                    │
│                                                                     │
│  // 4. 사용자 할당량 설정                                              │
│  $ sudo edquota username                                           │
│  $ sudo edquota -g groupname          // 그룹                        │
│  $ sudo edquota -p refuser newuser    // 참조 사용자 복사             │
│                                                                     │
│  // 5. 유예 기간 설정                                                 │
│  $ sudo edquota -t                                                  │
│                                                                     │
│  【할당량 확인】                                                       │
│  ──────────────────                                                │
│  // 사용자 할당량 확인                                                  │
│  $ quota                                                            │
│  $ quota -s                          // 인간 읽기 쉬운 형태            │
│  $ sudo quota username                                             │
│                                                                     │
│  // 전체 할당량 보고서                                                 │
│  $ sudo repquota /home                                              │
│  $ sudo repquota -a                  // 전체                         │
│  $ sudo repquota -s /home            // 요약                        │
│  $ sudo repquota -g /home            // 그룹                        │
│                                                                     │
│  // 상태 확인                                                        │
│  $ sudo quotaon -p /home             // 상태 확인                    │
│  $ quotaoff /home                    // 비활성화                      │
│                                                                     │
│  【XFS 할당량 설정】                                                   │
│  ──────────────────                                                │
│  // 마운트 옵션                                                       │
│  $ sudo mount -o uquota,gquota /dev/sda1 /home                     │
│  // uquota=사용자, gquota=그룹, pquota=프로젝트                       │
│                                                                     │
│  // 할당량 설정                                                       │
│  $ sudo xfs_quota -x -c 'limit bsoft=1g bhard=2g username' /home   │
│  $ sudo xfs_quota -x -c 'limit isoft=1000 ihard=2000 username' /home│
│                                                                     │
│  // 그룹 할당량                                                        │
│  $ sudo xfs_quota -x -c 'limit -g bsoft=5g bhard=10g groupname' /home│
│                                                                     │
│  // 확인                                                             │
│  $ sudo xfs_quota -x -c 'report -h' /home                          │
│  $ sudo xfs_quota -x -c 'quota -h username' /home                  │
│                                                                     │
│  // 프로젝트 할당량                                                     │
│  $ sudo xfs_quota -x -c 'project -s -p /home/project 1' /home      │
│  $ sudo xfs_quota -x -c 'limit -p bhard=10g 1' /home               │
│                                                                     │
│  【할당량 명령어 요약】                                                 │ |
│  ──────────────────                                                │
│  quotacheck   - 할당량 파일 생성/갱신                                  │
│  quotaon      - 할당량 활성화                                         │
│  quotaoff     - 할당량 비활성화                                        │
│  edquota      - 할당량 편집                                           │
│  quota        - 할당량 확인                                           │
│  repquota     - 할당량 보고서                                         │
│  setquota     - 명령행에서 할당량 설정                                  │
│  warnquota    - 초과 사용자에게 경고                                    │
│                                                                     │
│  【setquota 사용】                                                    │
│  ──────────────────                                                │
│  // 명령행에서 직접 설정                                                │
│  $ sudo setquota -u username 90000 100000 0 0 /home                │
│  //       [user] [block-soft] [block-hard] [inode-soft] [inode-hard]│
│                                                                     │
│  $ sudo setquota -g groupname 500000 600000 0 0 /home              │
│                                                                     │
│  // 유예 기간 설정                                                     │
│  $ sudo setquota -t 604800 604800 /home    // 7일 (초)               │
│                                                                     │
│  【Btrfs 할당량 (qgroup)】                                            │ |
│  ──────────────────                                                │
│  // qgroup 활성화                                                     │
│  $ sudo btrfs quota enable /mnt/btrfs                               │
│                                                                     │
│  // qgroup 생성                                                       │
│  $ sudo btrfs qgroup create 1/100 /mnt/btrfs                       │
│                                                                     │
│  // 서브볼륨에 qgroup 할당                                              │
│  $ sudo btrfs qgroup assign 0/256 1/100 /mnt/btrfs                 │
│                                                                     │
│  // 할당량 설정                                                        │
│  $ sudo btrfs qgroup limit 1G 1/100 /mnt/btrfs                     │
│                                                                     │
│  // 확인                                                             │
│  $ sudo btrfs qgroup show /mnt/btrfs                               │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 사용자/그룹별 디스크 사용량 제한
• 종류: 사용자 할당량, 그룹 할당량, 프로젝트 할당량
• 제한 단위: 블록(공간), inode(파일 수)
• Soft Limit: 경고 임계값, 유예 기간 있음
• Hard Limit: 절대 최대값, 초과 불가
• Grace Period: 유예 기간 (기본 7일)
• ext4: quota, quotacheck, edquota
• XFS: xfs_quota
• Btrfs: btrfs qgroup
• 활성화: mount -o usrquota,grpquota
• 확인: quota, repquota
• 설정: edquota, setquota
```
