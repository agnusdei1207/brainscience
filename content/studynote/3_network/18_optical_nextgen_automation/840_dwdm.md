# DWDM (Dense Wavelength Division Multiplexing)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: DWDM (Dense Wavelength Division Multiplexing)은 하나의 광섬유上に 여러 波長(파장)의 빛을 동시에 전송하여 전송 용량을 비약적으로 증가시키는 광전송 기술이다.
> 2. **가치**: 단일 광섬유에서 수십~수백 Gbps에서 수 Tbps로 전송 용량을 증가시켜, 장거리 및 대용량 통신 인프라의 대역폭 병목을 해소한다.
> 3. **융합**: 코히어렌트 디텍션, EDFA (Erbium-Doped Fiber Amplifier), 그리고 ROADM (Reconfigurable Optical Add-Drop Multiplexer)과의 결합을 통해 100Gbps 이상의 전송 속도와 동적 파장 할당이 가능해졌다.

---

## Ⅰ. 개요 및 필요성 (Context & Necessity)

### 개념 정의

DWDM (Dense Wavelength Division Multiplexing)은 광섬유 통신에서 사용되는 다중화 기법으로, 여러 波長(파장)의 레이저 빛을 하나의 광섬유에 함께 보내어 전송 용량을 크게 늘리는 기술이다. 각 파장을「채널」로 사용하여, 이론적으로는 단일 모드 광섬유 하나에 수십 개에서 수백 개의 파장을 추가할 수 있다. DWDM은 ITU-T G.694.1 표준에 의해 정의되어, C-밴드 (1530~1565nm) 내에 50GHz 또는 100GHz 간격으로 파장을 배치한다.

### 왜 DWDM이 필요한가

인터넷 트래픽은 매년 기하급수적으로 증가하고 있으며, 이에 따라 광섬유 네트워크의 전송 용량에 대한 수요도 크게 증가하고 있다. 새로운 광섬유 케이블을铺设하는 것은 비용이 매우 높고 시간도 오래 걸리므로, 기존 광섬유 인프라의 전송 용량을 극대화하는 기술이 필요했다. DWDM은 이미 설치된 광섬유 위에 추가적인 장치만으로 전송 용량을 수십 배 늘릴 수 있어, 비용 효율적인 해결책이 된다.

전통적인 광섬유 전송 vs DWDM의 차이를 비교하면, 용량 확장의 방식이根本적으로 다르다.

```text
  ┌─────────────────────────────────────────────────────────────────────┐
  │              전통적 광섬유 전송 vs DWDM 비교                              │
  ├─────────────────────────────────────────────────────────────────────┤
  │                                                                     │
  │  [전통적 광섬유 전송 (단일 파장)]                                    │
  │                                                                     │
  │  광원 ────▶ 단일 파장 (λ1) ────▶ 광섬유 ────▶ 수신기              │
  │                    │                                                │
  │                    │  전송 속도: 10 Gbps × 1 = 10 Gbps              │
  │                                                                     │
  │  문제: 더 높은 용량이 필요하면 새로운 광섬유를 설치해야 함              │
  │                                                                     │
  │  [DWDM (다중 파장)]                                                │
  │                                                                     │
  │  ┌─────────────────────────────────────────────────────────────┐   │
  │  │  λ1 (Ch1) ────────────────────────────────────────────▶  │   │
  │  │  λ2 (Ch2) ────────────────────────────────────────────▶  │   │
  │  │  λ3 (Ch3) ────────────────────────────────────────────▶  │   │
  │  │  ...                                                        │   │
  │  │  λN (ChN) ────────────────────────────────────────────▶  │   │
  │  └─────────────────────────────────────────────────────────────┘   │
  │                    │                                                │
  │                    │  전송 용량: 100 Gbps × 80 = 8 Tbps             │
  │                    │  (80파장, 각 100 Gbps 가정)                  │
  │                                                                     │
  │  ┌─────────────────────────────────────────────────────────────┐   │
  │  │  주요 구성 요소:                                              │   │
  │  │  - DWDM Mux: 여러 파장을 하나의 광섬유로 결합                  │   │
  │  │  - EDFA: 장거리 전송에서 신호 증폭                            │   │
  │  │  - DWDM Demux: 수신측에서 파장을 분리                        │   │
  │  │  - ROADM: 파장을 동적으로 추가/삭제 (Colorless, Directionless)│   │
  │  └─────────────────────────────────────────────────────────────┘   │
  └─────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 전통적 광섬유 전송에서는 하나의 파장(λ1)만 사용하므로, 전송 용량은 해당 파장의 전송 속도에 제한된다. 더 높은 용량이 필요하면 새로운 광섬유 케이블을 安装해야 하므로 비용과 시간이 많이 든다. DWDM에서는 여러 파장(λ1, λ2, λ3, ... λN)을 동시에 하나의 광섬유에 함께 전송한다. 각 파장은 독립적인 데이터 채널로 동작하며, 서로 간섭하지 않는다. DWDM Mux (Multiplexer)는 여러 파장을 하나의 광섬유로 결합하고, 수신측의 DWDM Demux (Demultiplexer)는 이를 다시 개별 파장으로 분리한다. EDFA (Erbium-Doped Fiber Amplifier)는 장거리 전송에서 모든 파장을 동시에 증폭하여, 각 파장마다 별도의 전기 신호 변환 없이도 신호를 증폭할 수 있게 해준다. ROADM (Reconfigurable Optical Add-Drop Multiplexer)은 특정 파장을 동적으로 추가/삭제할 수 있어, 네트워크 운영의 유연성을 크게 향상시킨다. 현재 상용화된 DWDM 시스템에서는 하나의 광섬유에 최대 96개(또는 그 이상)의 파장을 전송하며, 각 파장당 100Gbps 또는 200Gbps 속도를 지원하여, 총 전송 용량은 수십 Tbps에 달한다.

### 등장 배경

1. **인터넷 트래픽 폭발적 증가**: 2000년대 이후 인터넷 사용자와 데이터 소비가 급증하면서, 전송 용량에 대한 수요가 기하급수적으로 증가했다.
2. **광섬유 기술의 발전**: EDFA의 상용화와 단일 모드 광섬유의 확산으로, 장거리 광섬유 전송의 비용 효율성이 크게 향상됐다.
3. **ITU-T 표준화 (G.694.1)**: 2000년 ITU-T가 DWDM의 파장 배치를 표준화하여, 서로 다른厂商 장비 간의 상호운용성이 확보됐다.

### 💡 비유

DWDM은，就好像 여러颜色的光线(각 파장)를 하나의細いガラス管(광섬유) 안에 동시에通す 것과 같다. 각 색상은 서로 다른 정보를携带하고, 수신側에서各색光を分離하여元の情報を復元한다. 마치数十本の苍蝇가 단일관을 동시에通過하지만、서로 충돌하지 않는 것과 같다.

### 📢 섹션 요약 비유

 DWDM은，好像是一列の電車에서 各车厢(파장)이 각 다른 승객 정보(데이터)를 실어 나르고、열차 전체(광섬유)가 동시에進行하는 것과 같다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 구성 요소

| 요소명 | 역할 | 내부 동작 | 관련 기술 | 비유 |
|:---|:---|:---|:---|:---|
| **DWDM Mux** | 여러 파장을 하나의 광섬유로 결합 | 광대역 결합기 (Athermal Arrayed Waveguide Grating 등) | 광대역 결합기 | 입국 심사 게이트 |

| **EDFA (Erbium-Doped Fiber Amplifier)** | 모든 파장을 동시에 증폭 | 980nm 또는 1480nm 펌프 레이저로 유도 방출 | C-밴드 증폭 | 全駅停車駅 |
| **ROADM** | 특정 파장을 동적으로 추가/삭제 | WSS (Wavelength Selective Switch) 기반 | Colorless, Directionless CDC-ROADM | 교차점 톨게이트 |
| **파장 변환기 ( wavelength Converter)** | 파장 간 변환 | OEO (Optical-Electrical-Optical) 변환 | Full-range, Limited-range | 환승station |

**[다이어그램 해설]** 전송측에서 클라이언트 신호(SDH/SONET, Ethernet, Fibre Channel 등)는부터 OTU (Optical Transport Unit)에 입력된다. OTU는 클라이언트 신호를 ITU-T G.709 표준의 OTU 프레임으로 매핑하고, FEC (Forward Error Correction) 부호화를 적용하여 전송 중 발생할 수 있는 오류를 수신측에서修正할 수 있게 한다. 파장이 전송되면, DWDM Mux가 여러 파장을 하나의 광섬유로 결합한다. 이 과정에서는 Arrayed Waveguide Grating (AWG)나 Thin-Film Filter 등의 기구가 이용된다. 장거리 전송 구간에서는 약 80~100km마다 EDFA가 설치되어, 모든 파장을 동시에 증폭한다. EDFA는 희토류 원소인 erbium이 도핑된 광섬유로, 980nm 또는 1480nm 파장의 펌프 레이저로 에너지를 공급받아 C-밴드 전체를 동시에 증폭한다. 수신측에서는 DWDM Demux가 각 파장을 분리하고, OTU가 FEC를 복호화하여 클라이언트 신호를 역매핑한다. ROADM은 DWDM 네트워크에서 가장 중요한 혁신 중 하나로, 특정 파장을 동적으로 추가/삭제/전환할 수 있게 해준다. WSS (Wavelength Selective Switch) 기반의 ROADM은 1×N 포트로, 각 포트에 어떤 파장이든 할당할 수 있고(Colorless), 어떤 방향으로도 전송할 수 있다(Directionless). 이는 네트워크 운영의 유연성을 크게 향상시킨다.

### 📢 섹션 요약 비유

 DWDM의 동작 과정은，好像是一列の物流センター에서 各商品(파장)이 입고され(OTU 매핑)、물류 센터의全商品이 하나의 트럭에 함께積載되고(DWDM Mux)、중간 거점마다燃料补給(EDFA)을 하고、도착지에서各商品이 분리되어( Demux) 고객에게 전달되는 과정과 같다.

## Ⅲ. 융합 비교 및 다각도 분석

### 비교 1: DWDM vs CWDM (Coarse Wavelength Division Multiplexing)

| 비교 항목 | DWDM | CWDM |
|:---|:---|:---|
| **파장 수** | 40~96개 (고밀도) | 8~18개 (희소) |
| **파장 간격** | 50GHz 또는 100GHz (협대역) | 20nm (광대역) |
| **전송 거리** | 장거리 (80km+) | 단거리 (40~80km) |
| **증폭** | EDFA 필수 | EDFA 불필요 (저렴) |
| **비용** | 고가 | 저가 |
| **주 용도** | 통신사업자 코어, 데이터센터互联 | Metro 네트워크, 기업網 |

### 비교 2: 직렬 vs 병렬 전송

| 비교 항목 | 직렬 전송 (Time-Division) | 병렬 전송 (Wavelength-Division) |
|:---|:---|:---|
| ** 방식** | 시간 차례로 데이터 전송 | 여러 파장으로 동시에 전송 |
| **용량** | 제한적 | 대폭 증가 |
| ** 복잡성** | 단순 | 고도 (DWDM Mux/Demux 필요) |
| ** 상호接続性** | 어려움 (다중화 장치 필요) | 용이 (각 파장 독립 채널) |

### 과목 융합 관점

- **데이터센터互联 (DCI)**: DWDM은 데이터센터 간의 대용량 연결에 필수적인 기술이다. 400Gbps, 800Gbps Ethernet과 결합되어, 수십 km ~ 수백 km 거리의 데이터센터를 연결한다.
- **5G 전송网**: 5G 백홀(Backhaul) 네트워크에서 DWDM은 대용량数据传输의 핵심 기술로 사용된다. 5G의 높은 대역폭 요구사항을 만족하기 위해 100Gbps 이상의 DWDM 채널이 활용된다.

### 📢 섹션 요약 비유

 DWDM과 CWDM의 차이는，好像是一列의uo籠数의 차이와 같다. DWDM은 96개 이상의uo籠로 매우많은 차량을 동시에 운송하지만, 각uo籠 사이의 간격이 매우 좁아 정밀한 관리가 필요하다. CWDM은 8~18개의uo籠로 상대적으로 간격이 넓어 관리가 쉽지만, 동시에 운송할 수 있는 차량 수가 제한된다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 실무 시나리오

1. **시나리오 — 통신사업자 백본 네트워크 증설**: 기존 10Gbps DWDM 시스템의 용량이 부족해져, 100Gbps DWDM으로 업그레이드해야 하는 상황. 아키텍트는 기존 광섬유 인프라를 그대로 활용하고, DWDM 장비만 업그레이드하여 수일 만에 용량을 10배 확장했다.

2. **시나리오 — 데이터센터 간 400Gbps 연결**: 두 데이터센터 간에 400Gbps Ethernet 연결이 필요한 상황. 아키텍트는 400G DWDM 시스템을 도입하여, 4개의 100Gbps 파장을聚合하여 400Gbps 연결을実現했다.

### 도입 체크리스트

- **기술적**: 기존 광섬유의 전송 손실과 분산 특성이 DWDM 전송에 적합한가? EDFA 증폭 구간이 적절한가?
- **운영·보안적**: 파장 할당 계획(Channel Plan)이 수립되어 있는가? 장애 시 파장 복구 방안이 마련되어 있는가?

### 안티패턴

- **파장 드리프트**: 온도 변화나 장비 노후화로 인해 파장이 중심 파장에서 벗어날 수 있다. 이는 인접 파장에 간섭을 유발할 수 있으므로, 파장 안정성 모니터링이 필수적이다.
- **EDFA 증폭 균등화 부재**: EDFA가 모든 파장을 동일하게 증폭하지 않으면, 일부 파장의 신호 레벨이 낮아져 수신 에러율이 증가한다. 이득 평형(Gain Equalization)이 필요하다.

### 📢 섹션 요약 비유

 파장 드리프트는，好像是一列의uo籠数에서 各uo籠의 위치(파장)가列の進行中に少しずつずれていくことに似ている. 이것が激しいと、隣合ったuo籠が충돌하여cargo(데이터)가 손상될 수 있다.

## Ⅴ. 기대효과 및 결론

DWDM은 광섬유 전송 용량의 핵심 기술로, 기존 광섬유 인프라의 가치를 극대화하면서 수십 Tbps의 전송 용량을 제공한다. 코히어렌트 전송, 400Gbps/800Gbps Ethernet, 그리고 AI 기반 네트워크 최적화와 함께, DWDM은 미래 네트워크의 대역폭 요구사항을 충족하는 핵심 기술로 자리잡을 것이다.

### 📢 섹션 요약 비유

 DWDM의 미래은，好像是一列のuo籠数가 미래에는数千台のuo籠를 동시에 운송하는超级uo籠로 발전하는 것과 같다.

## 👶 어린이를 위한 3줄 비유 설명

1. DWDM은 마치 **무지개빛 광선 теле비전**와 같아서, 여러 색깔(각 파장)의 빛을 하나의細いガラス관(광섬유) 안에 동시에 보내면서 各색정보(데이터)를一緒に传输한다.
2. 만약 어떤 색깔의 빛이 섹이는(손실)되면, 중간에 있는 신호增幅장치(EDFA)가 모든 색깔을 함께增幅해서 다시 밝게 만들어줘요.
3. 이렇게 하면 한 개의細い유리관으로 수십 개의 무지개를 동시에 보낼 수 있어서, 엄청나게 많은 정보를 보낼 수 있어요!
