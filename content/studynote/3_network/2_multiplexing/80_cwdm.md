# CWDM (저밀도 파장 분할 다중화) 기술 분석

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파장 간격을 **20nm** 정도로 넓게 설정하여, 정밀한 온도 제어 장치 없이도 저렴하게 구현 가능한 광 다중화 방식.
> 2. **가치**: DWDM 대비 장비 가격이 매우 저렴하며, 주로 도시권 망(Metro)이나 데이터센터 연결과 같은 단거리(최대 80km) 구간에 최적화되어 있다.
> 3. **융합**: 광섬유의 OH- 피크(수산기 흡수 영역)를 극복한 'Full Spectrum CWDM' 기술을 통해 최대 18개 채널까지 확장하며 가성비 중심의 광 네트워크를 주도한다.

---

### Ⅰ. CWDM (Coarse WDM)의 핵심 매커니즘

- **작동 원리**: 
  1. 1271nm ~ 1611nm 대역에서 20nm 간격으로 파장을 할당.
  2. 파장 간격이 넓어 레이저의 파장 드리프트(Drift)를 허용함.
  3. 냉각 장치(Cooler)가 없는 저가형 레이저를 사용하여 구축 비용 절감.
- **특징**: ITU-T G.694.2 표준을 준수하며, 주로 8~16 채널을 사용함.

**📢 섹션 요약 비유**: 마치 고속도로 차선을 아주 넓게 그려서, 운전자가 조금 비틀거려도(파장 흔들림) 옆 차와 부딪히지 않게 설계한 안전하고 경제적인 도로와 같습니다.

---

### Ⅱ. CWDM 파장 배치 아키텍처 (ASCII)

넓은 간격을 통해 기술적 난이도를 낮춘 구조다.

```ascii
    [ CWDM Spectrum Map ]
    
    Power
      ^      CH 1        CH 2        CH 3
      |     +----+      +----+      +----+
      |     |    |      |    |      |    |
      |     |    |      |    |      |    |
    --+-----+----+------+----+------+----+------> Wavelength (nm)
            |    |      |    |      |    |
           1471  |     1491  |     1511  |
                 <---- 20nm ---->
                 (Wide Guard Band)
```

- **Wide Interval**: 파장 사이의 간격이 넓어 정밀한 필터나 온도 제어가 불필요함.
- **Uncooled Laser**: 냉각 장치가 없어 전력 소모가 적고 장비 크기가 작음.

**📢 섹션 요약 비유**: 마치 주차 공간을 아주 넓게 만들어 초보 운전자도 주차하기 쉽게 만든 대형 마트 주차장과 같습니다.

---

### Ⅲ. CWDM vs DWDM 경제성 및 기술 비교

| 비교 항목 | CWDM (Coarse) | DWDM (Dense) |
|:---|:---|:---|
| **파장 간격** | **20nm** | 0.8nm 이하 |
| **냉각 장치** | **불필요 (Uncooled)** | 필수 (TEC Cooler) |
| **최대 채널** | 18 채널 | 80~160 채널 |
| **전송 거리** | 50~80km (단거리) | 수천 km (장거리) |
| **비용(CAPEX)** | **매우 낮음** | 높음 |

**📢 섹션 요약 비유**: 경제적인 일반 우편(CWDM)과, 아주 정밀하고 빠르게 배달되지만 비용이 비싼 특급 배송(DWDM)의 차이와 같습니다.

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

#### 1. 메트로 구간 (Metro Network) 가성비 설계
- **현상**: 통신국사와 인접한 대형 빌딩 간의 전송 용량이 부족함.
- **기술사적 결단**: 
  - 수천만 원의 DWDM 장비 대신 수백만 원대의 **CWDM (Coarse Wavelength Division Multiplexing)** 장비를 도입한다. 
  - 이를 통해 10Gbps 광 링크를 8개 이상 즉시 확보하여, 예산 내에서 최대의 성능을 이끌어낸다.

#### 2. 기술사적 인사이트: 수산기(Water Peak) 문제
- 과거에는 1383nm 부근의 수산기 흡수 현상 때문에 CWDM 채널 사용에 제약이 있었다. 하지만 현대의 저수산기 광섬유(LWP; Low Water Peak) 보급으로 전 대역을 활용할 수 있게 되었음을 기술 제안 시 고려해야 한다.

**📢 섹션 요약 비유**: 마치 비가 오면 물웅덩이가 생겨 못 다니던 길(수산기 피크)을, 배수 공사(LWP 광섬유)를 통해 사계절 내내 다닐 수 있게 된 것과 같습니다.

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

#### 1. 정량/정성 기대효과
- **초기 구축비 절감**: DWDM 대비 약 1/3~1/5 수준의 비용으로 구축 가능.
- **운영 효율성**: 저전력, 저발열 장비로 유지보수 편의성 증대.

#### 2. 미래 전망
CWDM은 5G 프런트홀(Fronthaul) 구간에서 기지국과 안테나를 연결하는 핵심 기술로 재조명받고 있다. 특히, 파장 가변 레이저 기술과 결합하여 설치 편의성을 높인 'Self-Tunable CWDM'으로 발전하며, 도심 곳곳을 연결하는 실핏줄 같은 광 네트워크의 주인공이 될 것이다.

**📢 섹션 요약 비유**: 화려한 슈퍼카보다는 좁은 골목길 구석구석을 누비며 많은 짐을 나르는 경제적인 배달 오토바이처럼, 도심 통신의 없어서는 안 될 실속파 일꾼과 같습니다.
