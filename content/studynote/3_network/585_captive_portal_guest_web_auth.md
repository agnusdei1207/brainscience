# 585. 캡티브 포털 (Captive Portal) - 게스트 웹 인증

> **핵심 인사이트**: 공항이나 호텔에서 무료 와이파이를 잡고 사파리나 크롬을 열면, 내가 가고 싶은 네이버는 안 뜨고 갑자기 "약관에 동의하십니까?" 혹은 "호텔 룸 넘버를 치세요"라는 엉뚱한 웹페이지에 갇혀버린다. 이처럼 사용자를 강제로 특정 웹페이지에 가둬서 인증을 받아내는 낚시망 같은 시스템이 캡티브 포털이다.

## Ⅰ. 캡티브 포털(Captive Portal)의 개념
공공장소(공항, 호텔, 스타벅스 등)에서 개방형 무선 LAN(Wi-Fi)이나 유선망에 접속한 사용자가, **정상적인 인터넷을 사용하기 전에 강제적으로 특정 웹 페이지(인증/안내 페이지)로 리다이렉트(Redirect)되어, 약관 동의나 로그인, 요금 결제 등을 수행하도록 만드는 네트워크 보안 및 마케팅 기법**입니다.

## Ⅱ. 캡티브 포털의 동작 메커니즘 (어떻게 사람을 가두는가?) 🌟
와이파이를 잡은 사용자를 낚아채는 방법은 크게 3가지 계층의 기술을 섞어 씁니다.

### 1. HTTP 리다이렉트 (가장 고전적)
사용자가 브라우저를 열고 `http://www.naver.com`을 입력하면, 공유기(방화벽)가 이 HTTP GET 요청을 가로채어, `http://호텔인증.com`으로 가라는 상태 코드(HTTP 302 Found)를 던져 강제로 창을 돌려버립니다.

### 2. DNS 하이재킹 (DNS Hijacking)
사용자가 `naver.com`의 IP를 물어보려고 DNS 서버에 질의하면, 공유기가 중간에 이 질문(UDP 53)을 가로채서 진짜 네이버 IP 대신 **자신의 인증용 웹서버 IP 주소**를 가짜로 알려줍니다. 사용자는 네이버에 들어간 줄 알지만 실제로는 인증 페이지가 뜹니다.

### 3. IP/MAC 주소 기반 방화벽 차단
가장 확실한 방법입니다. 공유기(라우터)는 아직 인증 페이지에서 버튼(동의)을 누르지 않은 사용자의 MAC 주소를 기억해 두었다가, 이 기기가 보내는 **80번(HTTP), 443번(HTTPS) 포트 외의 모든 통신(카카오톡, 게임, 이메일)을 방화벽단에서 100% 차단(Drop)**해 버립니다. 오직 인증 페이지 서버로 가는 패킷만 허용(Whitelist)합니다.

## Ⅲ. HTTPS 도입으로 인한 캡티브 포털의 위기와 해결책
- **문제 발생**: 요즘은 거의 모든 사이트가 `https://` (암호화)를 씁니다. 공유기가 사용자의 `https://www.google.com` 요청을 가로채서 `http://호텔인증.com`으로 돌려버리면, 브라우저는 "누가 구글 인증서를 위조해 해킹을 시도한다!"라고 생각하고 새빨간 경고창을 띄우며 접속을 막아버려, 사용자가 영원히 인증 페이지를 보지 못하는 갇힘 현상이 발생했습니다.
- **해결책 (OS 차원의 탐지)**: 이를 해결하기 위해 요즘 스마트폰(iOS, Android)은 와이파이를 잡자마자 백그라운드에서 `http://captive.apple.com` 같은 아주 단순한 텍스트 페이지에 몰래 접속을 시도해 봅니다. 만약 "Success"라는 글씨가 안 오고 이상한 호텔 페이지가 뜨면, 아이폰이 "아, 여기 캡티브 포털 망이구나!"라고 눈치채고, 즉시 화면 위로 자체적인 브라우저(인증 창)를 팝업시켜 사용자가 동의를 누르게 유도합니다.

## Ⅳ. 주요 활용처
1. **공항/지하철 무상 Wi-Fi**: "무료 와이파이 이용 시 15초 광고 시청"을 강제하는 마케팅 용도.
2. **호텔/병원**: 객실 번호나 환자 번호를 입력하게 하여 외부인 접속을 차단.
3. **기업 내 방문자망(Guest Net)**: 외부 손님이 사내망(사내 서버)은 보지 못하고 인터넷만 쓰도록, 방문자 연락처를 적게 한 뒤 격리된 망을 열어줌.

> 📢 **섹션 요약 비유**: 캡티브 포털은 고속도로 진입로의 톨게이트 차단기입니다. 차(데이터)를 몰고 고속도로(인터넷)로 진입하려 하면 모든 길이 거대한 바리케이드로 막혀 있고 오직 '요금 정산소(인증 웹페이지)'로 빠지는 샛길만 열려 있습니다. 정산소에서 표를 뽑거나(약관 동의), 요금을 내야만(로그인), 비로소 톨게이트 차단기가 열려 자유롭게 부산(원하는 웹사이트)까지 달릴 수 있습니다.
