# 81. 고밀도 파장 분할 다중화 (DWDM, Dense WDM)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: DWDM(Dense WDM)은 CWDM의 널널했던 파장 간격을 0.8nm, 0.4nm 단위로 미친 듯이 촘촘하게(Dense) 썰어서, **광섬유 1가닥에 80개에서 최대 160개 이상의 초고속 레이저 빛을 쑤셔 넣는 인류 광통신의 끝판왕 아키텍처**다.
> 2. **기술적 한계 돌파 (쿨링과 앰프)**: 이렇게 촘촘하게 붙여 놓으면 기계 온도가 1도만 변해도 빛이 섞여버린다(간섭). 그래서 DWDM은 값비싼 열전소자(Peltier)로 레이저 온도를 소수점까지 얼려버리고, 수천 km를 가기 위해 EDFA라는 광 증폭기로 빛 덩어리를 한 번에 뻥튀기하는 극강의 하드웨어 기술을 동원한다.
> 3. **실무 융합**: 대서양을 횡단하는 해저 광케이블이나, SKT, KT 같은 1티어 통신사들의 **국가 단위 전국망 백본(Core Network)**은 100% 이 DWDM 장비로 돌아가고 있다.

---

## Ⅰ. DWDM의 개요 및 필요성 (Context & Necessity)

- **개념**: 
  - 광섬유의 손실이 가장 적은 $1550\text{nm}$ (C-Band) 대역을 집중 공략하여, 파장($\lambda$)의 간격을 $0.8\text{nm}$ (100GHz) 또는 $0.4\text{nm}$ (50GHz)로 극단적으로 좁게 분할해 80개 이상의 채널을 다중화하는 시스템.
- **필요성 (CWDM의 한계)**:
  - 1가닥에 18개 채널만 쏘는 CWDM으로는 쏟아지는 스마트폰(LTE/5G) 트래픽과 넷플릭스 4K 스트리밍을 도저히 감당할 수 없었다.
  - 게다가 CWDM은 50km만 넘어가면 빛이 소멸해 버리는데, 빛을 다시 살려줄 '광 증폭기'를 달 수 없는 치명적 단점이 있었다.
  - 결국 엔지니어들은 돈이 수억 원이 깨지더라도, **"빛을 소수점 단위로 칼같이 통제해서 100개를 밀어 넣고, 중간에 광 앰프로 수천 km를 쏘아버리자!"**라는 극한의 백본 아키텍처를 설계했고 이것이 DWDM이다.

- **💡 비유**: **DWDM은 '나노 스케일의 초정밀 주차장'**과 같다.
  - **CWDM (구형)**: 주차 칸이 엄청 넓어서 눈 감고 주차해도 옆 차를 안 긁는다. 하지만 땅에 차를 18대밖에 못 댄다.
  - **DWDM**: 주차 칸 간격이 단 **0.8mm**다. 바람만 불어 차가 흔들려도 옆 차와 대형 사고가 난다. 그래서 모든 차에 초정밀 냉각 센서와 위치 보정 AI를 강제로 달아, 차가 1mm도 움직이지 않게 얼려버린다. 덕분에 똑같은 땅에 차를 **160대**나 주차할 수 있게 되었다.

## Ⅱ. 핵심 아키텍처 및 메커니즘 (Deep Dive)

### 1. DWDM의 주파수 스펙트럼 맵핑

```text
  ┌────────────────────────────────────────────────────────────────────────┐
  │                 [DWDM의 극단적으로 촘촘한 파장 스펙트럼]                   │
  ├────────────────────────────────────────────────────────────────────────┤
  │                                                                        │
  │     광 파워(dBm)                                                        │
  │      ▲       (0.8 nm의 살인적인 간격)         (조금만 흔들려도 간섭 터짐)       │
  │      │   /\   /\   /\   /\   /\   /\   /\   /\   /\   /\   /\        │
  │      │  /  \ /  \ /  \ /  \ /  \ /  \ /  \ /  \ /  \ /  \ /  \       │
  │   ───┼┴────X────X────X────X────X────X────X────X────X────X────┴──▶ 파장 │
  │       1550.1   1550.9  1551.7  1552.5  1553.3  ...             (nm)│
  │                                                                        │
  │ * [핵심 기술]:                                                           │
  │   1. Cooled Laser : 온도 변화에 따른 파장 떨림(Drift)을 막기 위해 냉동기 장착  │
  │   2. EDFA (광 증폭기) : 저렇게 촘촘한 80개의 산봉우리를 한 번에 통째로 증폭시킴  │
  └────────────────────────────────────────────────────────────────────────┘
```

### 2. 구원자: EDFA (Erbium-Doped Fiber Amplifier)
과거에는 광케이블로 50km를 가다가 빛이 약해지면, 빛을 전기로 바꾸고(O/E), 전기를 증폭한 다음, 다시 전기를 빛으로 바꾸는(E/O) 멍청하고 느린 리피터를 썼다.
- **EDFA의 기적**: 에르븀(Erbium)이라는 희토류를 바른 유리에 펌프 레이저를 쏘면, 빛을 전기로 바꾸지 않고 **'날아가는 빛의 상태 그대로' 덩치를 100배 키워주는 아날로그 광 증폭기**가 탄생했다.
- **DWDM과의 찰떡궁합**: EDFA는 오직 1530~1565nm 대역의 빛만 증폭한다. DWDM은 아예 이 대역에만 80개의 파장을 몰빵해놨기 때문에, **EDFA 하나만 달면 80개의 채널이 한 방에 뻥튀기**되어 10,000km 해저 케이블을 횡단할 수 있게 되었다.

## Ⅲ. 실무 융합 및 트러블슈팅

### 실무 시나리오

1. **시나리오 — DWDM 장비의 항온항습 실패로 인한 통신망 전면 마비**:
   한여름, 부산 데이터센터의 에어컨이 고장 나 서버실 온도가 35도까지 치솟았다. 10분 뒤, 서울과 연결된 DWDM 백본망의 80개 채널 전체에 일제히 CRC Error가 쏟아지며 국가 전산망이 죽어버렸다.
   **[해결책]** **레이저의 열팽창에 의한 파장 드리프트(Wavelength Drift)** 대참사다. 광 트랜시버(SFP/XFP) 안에 있는 레이저 다이오드는 온도가 1도 오를 때마다 파장이 약 $0.1\text{nm}$씩 옆으로 밀린다. DWDM은 채널 간격이 불과 $0.8\text{nm}$다. 온도가 5도 오르자 1번 채널의 파장이 옆으로 밀리며 2번 채널을 덮치고, 2번이 3번을 덮치는 도미노 간섭(Crosstalk)이 터진 것이다. 긴급 조치로 장비 내부의 TEC(열전 냉각기) 팬을 풀가동시키고 이동식 에어컨을 투입해 장비 온도를 20도 이하로 억지로 얼려야 파장이 제자리로 돌아온다. (이것이 코어망 스위치에 항상 무식한 팬이 달려있는 이유다).

2. **시나리오 — 장거리 해저 광케이블의 '비선형성(Non-linear Effect)' 왜곡**:
   DWDM에 EDFA 앰프를 너무 많이 엮어서 태평양 밑으로 빛을 쐈다. 도착한 신호가 너무 약할까 봐 송신단에서 레이저 출력을 허용치(+20dBm) 이상으로 무식하게 세게 밀어 넣었더니, 오히려 수신단에서 데이터가 완전히 박살 난 채로 도착했다.
   **[해결책]** 광섬유의 **비선형성(Four-Wave Mixing, FWM)** 에러다. 좁은 유리관(코어) 안에 너무 강한 출력의 빛 80가닥을 동시에 때려 넣으면, 빛들끼리 에너지를 주고받으며 수학적으로 존재하지 않아야 할 '새로운 가짜 빛(유령 파장)'을 만들어낸다. 이 가짜 빛이 원래 데이터 채널과 겹치면서 원본을 파괴해 버린 것이다. DWDM 엔지니어는 절대 송신 파워를 무식하게 올리면 안 된다. 차라리 파워는 낮추되 중간에 EDFA 증폭기를 더 촘촘히 설치하여 '부드럽고 약하게 여러 번' 밀어주는 설계로 수정해야 한다.

## Ⅳ. 시스템 비교 및 최신 동향

### CWDM vs DWDM 아키텍처 비교 요약

| 지표 | CWDM (보급형) | DWDM (최고급형) |
|:---|:---|:---|
| **채널 간격** | 넓음 ($20\text{nm}$) | 미친 듯이 좁음 ($0.8\text{nm}$ 이하) |
| **채널 수 (속도)** | 최대 18개 (수십 Gbps) | 80~160개 이상 (수십 Tbps) |
| **광 증폭 (EDFA)**| **불가능** (빛 다 타버림) | **가능** (수천 km 장거리 전송의 핵심) |
| **쿨링 시스템** | Uncooled (쿨러 없음, 전기 적게 먹음) | Cooled (무식한 펠티어 쿨러 강제 가동) |
| **단가 및 사용처** | 저렴함 / 기업망, 도심망, 기지국 프론트홀 | 초고가 / SKT, KT 통신사 백본망, 해저망 |

### 최신 진화: ROADM (Reconfigurable Optical Add-Drop Multiplexer)
- 과거에는 대전에서 1530nm 파장 1개만 딱 내리고 싶으면 장비의 유리를 직접 뜯어서 사람이 수동 필터로 오려내야 했다. 
- 지금은 **ROADM**이라는 스마트 장비가 도입되어, 마우스 클릭 한 번이면 거울(WSS, 파장 선택 스위치)이 물리적으로 각도를 착 틀어서 원하는 색깔의 빛만 대전으로 딱 떨어뜨려 주고 나머지는 그냥 통과시킨다. 소프트웨어 정의 광 네트워크(SDN)의 완성판이다.

## Ⅴ. 결론

DWDM(고밀도 광파장 다중화)은 현대 인류가 만들어낸 가장 거대하고 정밀한 정보의 고속도로다. 이 기술이 없었다면 구글, 아마존, 넷플릭스가 내뿜는 천문학적인 데이터를 바다 건너로 전송하는 것은 물리적으로 불가능했을 것이다. 0.1nm의 오차도 허용하지 않는 극저온의 정밀 레이저와, 빛을 빛의 상태로 뻥튀기하는 EDFA 증폭기의 융합은 통신 공학을 넘어 양자 물리학이 이룩한 가장 위대한 상용화 성공 사례다.
