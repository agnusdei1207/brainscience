# 177. HMD (Head Mounted Display) 시선 추적과 포비티드 렌더링

> **핵심 인사이트**: 4K 쌍안경(VR 헤드셋)에 360도 풀 3D 화면을 돌리면 어지간한 괴물 컴퓨터도 불타오른다. 애플 비전 프로 같은 초격차 HMD는 눈동자(시선)를 추적하여, "어차피 사람이 뚜렷하게 보는 건 눈동자 정중앙뿐"이라는 인간 시야의 맹점을 역이용해, 보고 있는 곳만 4K로 그리고 주변은 모자이크처럼 뭉개버리는(포비티드 렌더링) 기적의 눈속임을 쓴다.

## Ⅰ. HMD (Head Mounted Display)의 개념
VR(가상현실)과 AR(증강현실)을 체험하기 위해 사용자가 머리(눈 앞)에 착용하는 디스플레이 기기입니다. 
몰입감을 극대화하기 위해 눈과 액정 패널의 거리가 불과 수 센티미터밖에 되지 않으므로, 초고해상도 화질(4K, 8K)과 90Hz 이상의 극도로 부드러운 화면 주사율이 필수적입니다. 이로 인해 GPU(그래픽 처리 장치)의 엄청난 연산 부하가 발생합니다.

## Ⅱ. 아이 트래킹 (Eye Tracking, 시선 추적)
HMD 안쪽에 달린 적외선 카메라와 센서가 사용자의 **동공 움직임과 시선 방향을 1초에 수백 번씩 실시간으로 쫓아가는(추적하는) 기술**입니다.
- **용도 1**: 마우스나 컨트롤러 없이, 눈동자로 메뉴를 쳐다보고 손가락을 튕기는 것만으로 화면을 클릭하는 직관적 인터페이스(UI)를 제공합니다. (애플 비전 프로의 핵심 조작법)
- **용도 2**: 아바타의 눈동자를 사용자의 실제 눈동자처럼 움직이게 하여 메타버스 내 교감 능력을 높입니다.
- **용도 3**: (가장 중요한 용도) 아래 설명할 **포비티드 렌더링**의 좌표를 GPU에게 짚어줍니다.

## Ⅲ. 포비티드 렌더링 (Foveated Rendering) - GPU 구원자
포비티드(Foveated)는 인간 망막에서 초점이 맺히는 중심와(Fovea)에서 유래한 단어입니다. 인간의 눈은 시선이 집중되는 한가운데만 선명하게 보고, 주변부(말초 시야)는 흐릿하게 인식합니다.

```text
[ 화면 전체를 고해상도로 그릴 경우 (기존 방식) ] ➔ GPU 100% 혹사, 발열 폭발

[ 포비티드 렌더링 적용 (눈동자 추적 연동) ]
┌──────────────────────────────────────┐
│  (흐릿함)                            │
│           (적당히 선명)              │
│                  🎯 시선 중심점      │
│                (초고해상도 4K)       │
│                                      │
│                                      │
└──────────────────────────────────────┘
```
1. 아이 트래킹 센서가 사용자가 쳐다보는 화면 좌표(🎯)를 읽어냅니다.
2. GPU는 **시선이 머무는 중심 반경만 100% 풀 해상도로 완벽하게 렌더링(그리기)** 합니다.
3. 사용자가 어차피 선명하게 보지 못하는 **주변 배경 화면은 해상도를 1/4로 대폭 낮추어 대충 뭉개서 렌더링**합니다.
4. 시선이 이동하면 0.01초 만에 새로운 초점 영역을 다시 4K로 그립니다.

## Ⅳ. 기술의 기대 효과
이 완벽한 눈속임 기법을 통해 **GPU의 연산량과 렌더링 부하를 50% 이상 획기적으로 깎아냅니다.** 
덕분에 HMD 기기에서 열이 덜 나고, 무거운 배터리 크기를 줄일 수 있으며, 프레임 드랍으로 인한 'VR 멀미(사이버 시크니스)'를 완벽히 예방할 수 있습니다.

> 📢 **섹션 요약 비유**: 미술관의 거대한 캔버스에 그림을 그릴 때, 관람객이 서치라이트(시선)를 비추는 곳만 세계 최고의 화가(GPU)가 엄청난 디테일로 유화를 그리고, 불이 꺼진 어두운 주변은 붓으로 대충 칠해버리는 것입니다. 관람객이 라이트를 옮기면 그곳만 다시 순식간에 명작으로 그려내어, 미술관 전체가 엄청나게 선명하다는 '완벽한 착각'을 심어주는 마술입니다.
